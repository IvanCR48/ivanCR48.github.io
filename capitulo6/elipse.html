2.1 Concepto y Elementos de la Elipse

d(P,F1) + d(P,F2) = 2a

Históricamente se consideraba que las órbitas planetarias eran circulares, pero Kepler descubrió en 1610 que las trayectorias de los planetas son elípticas con el Sol en uno de sus focos.

Definición Formal como Lugar Geométrico

La elipse es el lugar geométrico de todos los puntos P(x,y) del plano tales que la suma de sus distancias a dos puntos fijos F1 y F2 (llamados focos) es constante e igual a 2a:

d(P, F1) + d(P, F2) = 2a     (con 2a > 2c)

• Focos (F1, F2): Puntos fijos sobre el eje focal. Distancia entre focos: F1F2 = 2c (semidistancia focal c).
• Centro C(h,k): Punto medio del segmento focal F1F2.
• Vértices Principales (V1, V2): Intersección con la recta focal. Longitud del Eje Mayor: V1V2 = 2a (semieje mayor a).
• Vértices Secundarios (B1, B2): Intersección con la recta secundaria. Longitud del Eje Menor: B1B2 = 2b (semieje menor b).
• Relación Pitagórica Fundamental:

a2 = b2 + c2     (donde a > b  y  a > c)

• Lado Recto (L.R.): Cuerda focal perpendicular al eje focal. Longitud: L.R. = 2b2 / a.

2.2 Valor de la Constante (2a) y Excentricidad (e)

Excentricidad de la Elipse

A toda elipse se le asocia un cociente que mide su grado de "achatamiento" u "ovalado", denominado excentricidad (e):

e = c / a     (con 0 < e < 1)

• Como c < a, el valor de la excentricidad para cualquier elipse cumple estrictamente: 0 < e < 1.
• Si e → 0: Los focos se acercan al centro (c → 0), los semiejes se igualan (a ≈ b) y la elipse tiende a una circunferencia.
• Si e → 1: La semidistancia focal se aproxima al semieje mayor (c → a), haciendo la elipse cada vez más alargada o achatada.

2.3 Ecuaciones con Centro en el Origen C(0,0)

Eje Focal Horizontal (sobre eje x)

x2 / a2 + y2 / b2 = 1   (a > b)

• Focos: F1(c, 0), F2(−c, 0)
• Vértices Mayores: V1(a, 0), V2(−a, 0)
• Vértices Menores: B1(0, b), B2(0, −b)

Eje Focal Vertical (sobre eje y)

x2 / b2 + y2 / a2 = 1   (a > b)

• Focos: F1(0, c), F2(0, −c)
• Vértices Mayores: V1(0, a), V2(0, −a)
• Vértices Menores: B1(b, 0), B2(−b, 0)

Elipse Canónica Horizontal centrada en el Origen C(0,0) con semiejes a y b

1(c,0) F2(−c,0) V1(a,0) V2(−a,0) B1(0,b) B2(0,−b)

2.4 Ecuaciones Principal y General para C(h,k)

Forma Principal (Canónica Trasladada)

Al aplicar una traslación T(h, k) al centro, la ecuación de la elipse resulta:

(x − h)2 / a2 + (y − k)2 / b2 = 1     (Eje focal horizontal, a > b)

(x − h)2 / b2 + (y − k)2 / a2 = 1     (Eje focal vertical, a > b)

Forma General y Criterio de Identificación

Desarrollando los binomios e igualando a cero se obtiene la Ecuación General:

A x2 + B y2 + C x + D y + F = 0

Criterio Clave para Identificar una Elipse:
Los coeficientes de los términos cuadráticos A y B deben ser DISTINTOS pero del MISMO SIGNO (A ≠ B y A · B > 0).
• Si A < B ⇒ El eje focal es horizontal (el mayor denominador a2 acompaña a x).
• Si A > B ⇒ El eje focal es vertical (el mayor denominador a2 acompaña a y).

2.5 Ejemplo Resuelto de Completación de Cuadrados

Dada: 5x2 + 9y2 − 80x + 54y + 221 = 0

1. Agrupamos por variables y pasamos el término independiente:
    (5x2 − 80x) + (9y2 + 54y) = −221.
2. Extraemos factor común de los coeficientes de x2 e y2:
    5(x2 − 16x) + 9(y2 + 6y) = −221.
3. Completamos trinomios dentro de los paréntesis:
    5(x2 − 16x + 64) + 9(y2 + 6y + 9) = −221 + 5(64) + 9(9)
    5(x − 8)2 + 9(y + 3)2 = −221 + 320 + 81 = 180.
4. Dividimos ambos miembros por 180 para obtener 1 a la derecha:
    (x − 8)2 / 36 + (y + 3)2 / 20 = 1.

Elementos Obtenidos:
• Centro: C(8, −3) (h=8, k=-3).
• Semiejes: a2 = 36 ⇒ a = 6 (mayor, horizontal)  |  b2 = 20 ⇒ b = 2√5 ≈ 4,47.
• Semidistancia focal c: c2 = a2 − b2 = 36 − 20 = 16 ⇒ c = 4.
• Focos: F1(8+4, −3) = F1(12, −3)  y  F2(8−4, −3) = F2(4, −3).
• Vértices Mayores: V1(8+6, −3) = V1(14, −3)  y  V2(8−6, −3) = V2(2, −3).
• Lado Recto: L.R. = 2b2 / a = 2(20) / 6 = 20/3 ≈ 6,67.
• Excentricidad: e = c / a = 4 / 6 = 2/3 ≈ 0,67.

2.6 Ejercicios Prácticos del Módulo

Ejercicio 250-i: x2/25 + y2/9 = 1

Indicar todos los elementos de la elipse x2/25 + y2/9 = 1.

Ver Elementos

• Centro: C(0,0).
• Semiejes: a = 5 (eje x), b = 3 (eje y).
• Focal: c = √(25 − 9) = √16 = 4.
• Focos: F1(4,0) y F2(−4,0).
• Vértices mayores: V1(5,0) y V2(−5,0).
• Excentricidad: e = 4/5 = 0,8.

Ejercicio 251-i: F1(0,2), F2(0,-2), a=4

Determinar la ecuación de la elipse dados sus focos y el semieje mayor a = 4.

Ver Resolución

1. Centro en el origen C(0,0) con eje focal vertical (sobre eje y).
2. Distancia focal c = 2.
3. Semieje menor: b2 = a2 − c2 = 16 − 4 = 12.
4. Ecuación Canónica: x2/12 + y2/16 = 1.

Ejercicio 253-i: C(8, -3), 2a = 12, e = 2/3

Hallar la ecuación de la elipse con C(8, −3), eje mayor 2a = 12, excentricidad e = 2/3 y eje focal horizontal.

Ver Resolución

1. Semieje mayor a = 6 \implies a^2 = 36.
2. Excentricidad: e = c / a = c / 6 = 2/3 \implies c = 4 \implies c^2 = 16.
3. Semieje menor: b^2 = a^2 - c^2 = 36 - 16 = 20.
4. Ecuación Principal: (x − 8)2 / 36 + (y + 3)2 / 20 = 1.
