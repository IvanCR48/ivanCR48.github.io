Definicion Algebraica

|a|

Muchas veces deseamos describir el tamaño de un numero sin importar si es positivo o negativo. Para hacerlo, introducimos el concepto de valor absoluto (o modulo).

Definicion formal

Sea a un numero real, su valor absoluto se denota por |a| y se define como:

Esta definicion en dos partes significa que: si el numero es positivo o cero, queda igual; si es negativo, se toma su opuesto (inverso aditivo), lo que da un resultado positivo.

Ejemplo: Base positiva

|7| = 7

Como a=7 es ≥ 0, el valor absoluto es el mismo numero.

Ejemplo: Base negativa

|-7| = -(-7) = 7

Como a=-7 es < 0, se le antepone un signo menos para hacerlo positivo.

Modulo de cero

El valor absoluto de cero es siempre cero: |0| = 0.

Interpretacion Geometrica

El modulo como distancia al origen

Geometricamente, la expresion |x| representa la distancia desde el punto x hasta el 0 (origen) en la recta real.

Recta real: Distancia al origen

Ecuaciones sencillas con valor absoluto

La ecuacion |x| = 4 representa todos los numeros reales cuya distancia al origen es exactamente 4. Por lo tanto, tiene dos soluciones:

x = 4     o     x = -4

Distancia entre dos puntos

d[a,b]

Definicion general de distancia

Dados dos numeros reales a y b, la distancia entre ellos en la recta numerica se denota por d[a;b] y coincide con el modulo de su diferencia:

d[a;b] = |a - b|

Distancia entre dos puntos cualesquiera

Si a < b

d[a;b] = b - a

Ejemplo: distancia entre 3 y 8.
d[3;8] = 8 - 3 = 5

Si a > b

d[a;b] = a - b

Ejemplo: distancia entre 8 y 3.
d[8;3] = 8 - 3 = 5

Curiosidad sobre el orden

Como la distancia es una magnitud positiva, no importa el orden en que restemos dentro del modulo:

|a - b| = |b - a|

Por ejemplo, la distancia entre 7 y -3 es:
d[7;-3] = |7 - (-3)| = |7 + 3| = |10| = 10
O bien:
d[-3;7] = |-3 - 7| = |-10| = 10

Propiedades del Valor Absoluto

Sean a y b numeros reales. Se cumplen las siguientes propiedades fundamentales:

N° Propiedad Formula Ejemplo
1 Modulo del producto |a · b| = |a| · |b| |(-3) · 4| = |-3| · |4| = 3 · 4 = 12
2 Modulo del cociente |a / b| = |a| / |b|   (b ≠ 0) |(-15) / 5| = |-15| / |5| = 15 / 5 = 3
3 Desigualdad triangular |a + b| ≤ |a| + |b| |(-3) + 5| = |2| = 2 ≤ |-3| + |5| = 3 + 5 = 8
4 Modulo del opuesto |-a| = |a| |-5| = |5| = 5
5 Cuadrado del modulo |a|2 = a2 |-3|2 = 32 = 9    y    (-3)2 = 9

Conexion con la raiz cuadrada

Existe una relacion sumamente importante entre el valor absoluto y la raiz cuadrada de un cuadrado:

√(a2) = |a|

Esto garantiza que el resultado de la raiz cuadrada sea siempre no negativo.
Ejemplo: √(62) = |6| = 6    y    √((-6)2) = |-6| = 6.

Ejemplos de Simplificacion

Escribir las siguientes expresiones sin los simbolos de valor absoluto:

Ejemplo A: |π - 3|

|π - 3| = π - 3

Como π ≈ 3.14159... es mayor que 3, la expresion interior π - 3 es positiva (≥ 0). Por lo tanto, sale del modulo sin cambios.

Ejemplo B: |3 - π|

|3 - π| = - (3 - π) = π - 3

Como 3 es menor que π, la expresion interior 3 - π es negativa. Por definicion, debemos cambiarle el signo anteponiendo un menos, obteniendo π - 3 (que es positivo).

Ejemplo C: |x6 + 1|

|x6 + 1| = x6 + 1

Cualquier numero real elevado a una potencia par es no negativo (x6 ≥ 0). Al sumarle 1, la expresion x6 + 1 es estrictamente positiva para cualquier x. Por lo tanto, se eliminan las barras directamente.
