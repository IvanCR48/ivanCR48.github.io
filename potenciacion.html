Potenciacion

an

Definicion

Si a es un numero real (a ≠ 0) y n es un numero natural, entonces:

an = a · a · a · ... · a    (n veces)

Se lee "a elevado a la n". El numero a se llama base y n se llama exponente.

Anatomia de una potencia

Convenciones importantes

Exponente 1

a1 = a

Cualquier numero elevado a 1 es el mismo numero.

Exponente 0

a0 = 1   (a ≠ 0)

Todo numero no nulo elevado a 0 es igual a 1.

Exponente negativo

a-n = 1 / an

Un exponente negativo equivale al reciproco de la potencia positiva.

Ejemplo concreto

3-2 = 1/32 = 1/9

Se invierte la base y se usa el exponente positivo.

Propiedades de la potenciacion

Sean a, b numeros reales no nulos, m y n numeros enteros.

Propiedad Formula Ejemplo
Producto de igual base an · am = an+m 23 · 24 = 27 = 128
Potencia de potencia (an)m = an·m (52)3 = 56 = 15625
Cociente de igual base an / am = an-m 75 / 72 = 73 = 343
Potencia de un producto (a · b)n = an · bn (2 · 3)2 = 4 · 9 = 36
Potencia de un cociente (a/b)n = an / bn (3/2)2 = 9/4

Visualizacion: producto de potencias de igual base

Error comun: la potencia NO distribuye sobre la suma

INCORRECTO

(a + b)n ≠ an + bn

(a - b)n ≠ an - bn

Ejemplo numerico

(1 + 2)2 = 32 = 9

12 + 22 = 1 + 4 = 5

9 ≠ 5

Tres productos notables para recordar

Cuadrado de una suma

(a + b)2 = a2 + 2ab + b2

(a + b)(a + b)

= a2 + ab + ba + b2

= a2 + 2ab + b2

Cuadrado de una diferencia

(a - b)2 = a2 - 2ab + b2

(a - b)(a - b)

= a2 - ab - ba + b2

= a2 - 2ab + b2

Comparacion

Misma estructura que (a+b)2 pero con el termino central negativo: -2ab en lugar de +2ab. El termino b2 siempre es positivo.

Diferencia de cuadrados

(a - b)(a + b) = a2 - b2

(a - b)(a + b)

= a2 + ab - ba - b2

= a2 - b2

Los terminos ab y -ba se cancelan.

Precauciones comunes

Errores frecuentes que hay que evitar al trabajar con potencias.

Signo y parentesis
Correcto (-2)4 = (-2)·(-2)·(-2)·(-2) = 16
Distinto -24 = -(24) = -16

Sin parentesis, el signo negativo no forma parte de la base. Solo se eleva el 2 y luego se aplica el signo negativo.

Potencia de potencia vs. Producto de potencias
(b4)2 = b4·2 = b8
b4 · b2 = b4+2 = b6

Potencia de potencia: se multiplican los exponentes.
Producto de igual base: se suman los exponentes.

Inverso de una suma
Incorrecto (a-1 + b-1)-1 = a + b
Correcto (a-1 + b-1)-1 = (1/a + 1/b)-1 = ab/(a+b)

El exponente -1 no se distribuye sobre la suma. Primero hay que operar dentro del parentesis.

Radicacion

√

Raiz cuadrada

Si a es un numero real positivo, √a es el numero real positivo b tal que:

√a = b    si y solo si    b2 = a    (b ≥ 0)

Raiz cubica

La raiz cubica esta definida para cualquier numero real (positivo o negativo):

3√a = b    si y solo si    b3 = a

Raiz enesima: regla general

n PAR

n√a = b   con a ≥ 0, b ≥ 0

Solo definida para numeros positivos (o cero). El resultado siempre es positivo.

4√(-16) no existe en R

n IMPAR

n√a = b   para todo a ∈ R

Definida para cualquier real, tanto positivo como negativo.

3√(-27) = -3

Ejemplos de radicacion

Expresion Resultado Verificacion
√9 3 32 = 9
√16 4  (no ±4) 42 = 16
3√64 4 43 = 64
√1 1 12 = 1
3√(-27) -3 (-3)3 = -27

Propiedades de la radicacion

Sean a, b numeros reales, n y m numeros naturales. Todas las operaciones deben estar definidas.

Propiedad Formula Ejemplo
Raiz de un producto n√(a·b) = n√a · n√b √(4·9) = √4 · √9 = 2 · 3 = 6
Raiz de un cociente n√(a/b) = n√a / n√b √(25/4) = √25 / √4 = 5/2
Raiz de raiz m√(n√a) = m·n√a √(√16) = 4√16 = 2

Potencia de exponente racional

am/n

Hasta ahora definimos an para exponentes enteros. Ahora extendemos la definicion para cuando el exponente es un numero racional (una fraccion m/n).

Paso 1: Exponente 1/n

Sea a ≥ 0 y n un entero mayor que 1. Definimos:

a1/n = n√a

El exponente 1/n equivale a tomar la raiz enesima.

Ejemplos de a^(1/n)

Expresion exponencial Como raiz Resultado
41/2 √4 2
271/3 3√27 3
161/4 4√16 2

Paso 2: Exponente m/n

Si m y n son enteros positivos y n > 1, definimos:

am/n = n√(am) = (n√a)m

Se puede primero elevar y luego extraer raiz, o al reves. Ambos caminos dan el mismo resultado.

Dos caminos equivalentes para calcular a^(m/n)

Ejemplos de a^(m/n)

Expresion Camino 1: elevar, luego raiz Camino 2: raiz, luego elevar Resultado
82/3 3√(82) = 3√64 (3√8)2 = 22 4
272/3 3√(272) = 3√729 (3√27)2 = 32 9
43/2 √(43) = √64 (√4)3 = 23 8

Paso 3: Exponente racional negativo

Combinando con la regla del exponente negativo:

a-m/n = 1 / am/n

Ejemplos de exponente racional negativo

Expresion Desarrollo Resultado
4-1/2 1 / 41/2 = 1 / √4 1/2
8-2/3 1 / 82/3 = 1 / (3√8)2 = 1 / 4 1/4
(-27)2/3 (3√(-27))2 = (-3)2 9

Base negativa con exponente racional

La definicion de am/n con a negativa solo funciona si n es impar. Por ejemplo:

(-27)2/3 = (3√(-27))2 = (-3)2 = 9   (n=3 impar, valido)

(-4)1/2 = √(-4)   no existe en R   (n=2 par, no valido)

Resumen: extension progresiva del exponente

1 an   exponente natural (n ∈ N)
2 a0 = 1,   a-n = 1/an   exponente entero (n ∈ Z)
3 am/n = n√(am)   exponente racional (m/n ∈ Q)

En cada extension, las propiedades de la potenciacion (producto de igual base, potencia de potencia, etc.) siguen funcionando. Esa es la razon de estas definiciones.

Atencion: √(a²) no siempre es a

Si a = -5, entonces:

(-5)2 = 25  ⇒  √((-5)2) = √25 = 5 ≠ -5

La raiz cuadrada siempre da resultado positivo. Por eso √(a2) = |a| (valor absoluto), no simplemente a.
