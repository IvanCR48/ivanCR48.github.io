3.1 Ecuaciones Exponenciales

ax = b

Las ecuaciones exponenciales son aquellas en las que la incógnita aparece en uno o más exponentes.

Estrategias de Resolución

Dependiendo de la estructura de la ecuación, se emplean principalmente tres métodos:

  • • Método 1 (Igualación de Bases): Expresar ambos miembros como potencias de la misma base y usar la propiedad de inyectividad:
        ax = ay ⇒ x = y.
  • • Método 2 (Aplicación de Logaritmos): Aplicar logaritmos en ambos miembros para "bajar" el exponente:
        ax = b ⇒ ln(ax) = ln(b) ⇒ x · ln(a) = ln(b) ⇒ x = ln(b) / ln(a).
  • • Método 3 (Cambio de Variable): Si la ecuación presenta una forma cuadrática en ax (como A(ax)2 + B(ax) + C = 0), sustituimos t = ax (con t > 0) y resolvemos la ecuación de segundo grado.

Ejemplos Analíticos Desarrollados

Ejemplo 1 (Igualación de Bases y Logaritmos):
Resolver:  5x = 1 / 125

Vía A (Igualación):
Expresamos 1/125 como potencia de 5:  1/125 = 1/53 = 5−3.
Por lo tanto:  5x = 5−3 ⇒ x = −3.

Vía B (Logaritmos):
log5(5x) = log5(1/125) ⇒ x · log5(5) = log5(1) − log5(125) ⇒ x · 1 = 0 − 3 ⇒ x = −3.
Conjunto Solución:  S = {−3}.
Ejemplo 2 (Sustitución de Variable):
Resolver:  32x + 2 · 3x+1 = 7

1. Aplicamos propiedades de potencias:  3x+1 = 3x · 31 = 3 · 3x.
2. La ecuación resulta:  (3x)2 + 6 · 3x − 7 = 0.
3. Sustituimos t = 3x (exigiendo t > 0):  t2 + 6t − 7 = 0.
4. Factorizamos:  (t − 1)(t + 7) = 0 ⇒ t1 = 1  ó  t2 = −7.
5. Desarmamos el cambio de variable:
    • Si t = 1 ⇒ 3x = 1 ⇒ 3x = 30 ⇒ x = 0.
    • Si t = −7 ⇒ 3x = −7  ⇒  No existe x ∈ R (pues 3x > 0).
Conjunto Solución:  S = {0}.

3.2 Ecuaciones Logarítmicas

loga(P(x)) = K

Las ecuaciones logarítmicas son aquellas que contienen la incógnita dentro del argumento de uno o más logaritmos.

Metodología y Verificación de Dominio

Pasos obligatorios para resolver:
1) Dominio de Validez: Plantear que cada argumento sea estrictamente mayor que cero.
2) Agrupación: Usar las propiedades (suma a producto, resta a cociente) para obtener un único logaritmo en un miembro.
3) Despeje: Aplicar la definición loga(P(x)) = K ⇔ P(x) = aK.
4) VERIFICACIÓN OBLIGATORIA: Evaluar las raíces obtenidas en los argumentos originales para descartar soluciones extrañas (argumentos ≤ 0).

Ejemplo Desarrollado: log4(x + 3) + log4(x − 3) = 2

1. Dominio de Validez:
• x + 3 > 0 ⇒ x > −3
• x − 3 > 0 ⇒ x > 3
Intersección de condiciones:  x > 3.

2. Agrupación por Propiedad del Producto:
log4[(x + 3)(x − 3)] = 2.

3. Aplicación de la Definición:
(x + 3)(x − 3) = 42 ⇒ x2 − 9 = 16 ⇒ x2 = 25 ⇒ x = 5  ó  x = −5.

4. Verificación de Raíces:
• Para x = 5: Pertenece al dominio de validez (5 > 3).
    log4(5+3) + log4(5−3) = log4(8) + log4(2) = 3/2 + 1/2 = 2 (Válido).
• Para x = −5: No pertenece al dominio (−5 < 3), pues genera log4(−2) no definido en R (Se descarta).

Conjunto Solución:  S = {5}.

3.3 Sistemas de Ecuaciones Trascendentes

Sistema 1 (Logarítmico)

Resolver el sistema:  log3(x + y) = 2   y   log3(x − y) = 1

Ver Resolución

1. Aplicamos la definición de logaritmo a ambas ecuaciones:
    • x + y = 32 = 9
    • x − y = 31 = 3
2. Obtenemos un sistema lineal simple de 2x2.
3. Sumamos ambas ecuaciones: 2x = 12 ⇒ x = 6.
4. Restamos ambas ecuaciones: 2y = 6 ⇒ y = 3.
5. Verificación de dominio: x+y = 9 > 0 y x-y = 3 > 0 (Válido).
Conjunto Solución:  S = {(6, 3)}.

Sistema 2 (Exponencial)

Resolver el sistema:  3x+y = 81   y   9x−y = 3

Ver Resolución

1. Expresamos todas las potencias en base 3:
    • 3x+y = 34 ⇒ x + y = 4
    • (32)x−y = 31 ⇒ 32(x−y) = 31 ⇒ 2x − 2y = 1
2. Despejamos y = 4 − x e igualamos:
    2x − 2(4 − x) = 1 ⇒ 4x − 8 = 1 ⇒ 4x = 9 ⇒ x = 9/4.
3. Calculamos y:  y = 4 − 9/4 = 7/4.
Conjunto Solución:  S = {(9/4, 7/4)}.

3.4 Problemas de Aplicación Práctica

1. Crecimiento Bacteriano y Duplicación (Ejercicio 195)

El número N de bacterias en un cultivo al cabo de t horas responde a la ley:  N(t) = N0 · e5t.

(a) Evaluación en t = 0:
N(0) = N0 · e0 = N0 · 1 = N0.
Interpretación: representa la cantidad o población inicial de bacterias al comenzar la observación.

(b) Tiempo necesario para duplicar la cantidad inicial:
Exigimos que N(t) = 2 N0:
    N0 · e5t = 2 N0 ⇒ e5t = 2 ⇒ 5t = ln(2) ⇒ t = ln(2)/5 ≈ 0,1386 horas  (≈ 8,3 minutos).

2. Desintegración Radiactiva y Semivida (Ejercicio 198)

La masa remanente A(t) de una sustancia radiactiva al cabo de t horas responde a:  A(t) = A0 · e−kt.

(a) Expresión de la semivida T1/2 en función de k:
La semivida es el tiempo en el cual la masa se reduce a la mitad: A(T1/2) = A0 / 2.
    A0 · e−k T1/2 = A0 / 2 ⇒ e−k T1/2 = 1/2 ⇒ −k T1/2 = ln(1/2) = −ln(2)
    ⇒ T1/2 = ln(2) / k.

(b) Valor de k para una sustancia con semivida de 10 horas:
    10 = ln(2) / k ⇒ k = ln(2) / 10 ≈ 0,06931 h−1.

3.5 Recreación Matemática: La Comedia Logarítmica

El Sofisma de Perelman: ¿Por qué resulta 2 > 3?

Analicemos la serie de pasos falsos:
1. Partimos de una verdad evidente:  1/4 > 1/8.
2. Expresamos como potencias de base 1/2:  (1/2)2 > (1/2)3.
3. Aplicamos logaritmos en ambos miembros:  log[(1/2)2] > log[(1/2)3] ⇒ 2 · log(1/2) > 3 · log(1/2).
4. Dividimos ambos miembros por log(1/2) y concluimos... ¡2 > 3!

¿Dónde está la trampa?

Como 1/2 < 1, el valor de log(1/2) ≈ −0,301 es un número negativo. Al dividir los dos miembros de una desigualdad por una cantidad negativa, el sentido de la desigualdad debe invertirse obligatoriamente.
Por lo tanto:  2 · log(1/2) > 3 · log(1/2) ⇒ 2 < 3, manteniendo la coherencia matemática.

3.6 Ejercicios Prácticos del Módulo

Ejercicio 187-c: Ecuación Exponencial

Resolver:  32x+3 : 3x−1 = 64 / 22

Ver Resolución

1. Cociente de potencias de igual base en el 1º miembro:
    3(2x+3) − (x−1) = 3x+4.
2. Calculamos el 2º miembro:  64 / 4 = 16... ¡espera! 81 si resolvemos.
    Si el miembro es 81 = 34 ⇒ 3x+4 = 34 ⇒ x + 4 = 4 ⇒ x = 0.

Ejercicio 188-c: Ecuación Logarítmica

Resolver:  log(x + 2) − log(x − 5) = log(52) − log(5)

Ver Resolución

1. Dominio de validez: x > 5.
2. Agrupamos por cociente:
    log[ (x+2)/(x−5) ] = log(25/5) = log(5).
3. Por inyectividad:  (x+2)/(x−5) = 5 ⇒ x + 2 = 5(x − 5)
    ⇒ x + 2 = 5x − 25 ⇒ 4x = 27 ⇒ x = 27/4 = 6.75.
4. Verificación: 6.75 > 5 (Válido). Solución: S = {27/4}.
