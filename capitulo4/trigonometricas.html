5.1 Ángulos Orientados y Sistemas de Medición

α° ⇔ R rad

En trigonometría, un ángulo se genera mediante la rotación de una semirrecta alrededor de su origen. El sentido de giro determina su orientación.

Orientación y Congruencia

• Sentido Positivo (+): Rotación en sentido antihorario (contrario a las agujas del reloj).
• Sentido Negativo (−): Rotación en sentido horario (igual a las agujas del reloj).
• Ángulos Congruentes: Coinciden en sus lados inicial y terminal. Su diferencia es un número entero de vueltas completas:

α ≡ β  ⇔  α − β = k · 360°   ó   α − β = k · 2π     (con k ∈ Z)

Sistemas Sexagesimal y Circular (Radianes)

• Sistema Sexagesimal: Unidad principal es el grado sexagesimal (1° = 1/360 de vuelta completa). 1° = 60' (minutos) y 1' = 60'' (segundos).
• Sistema Circular (Radianes): Un radián (rad) es la medida del ángulo central que subtiende un arco de longitud igual al radio de la circunferencia. Una vuelta completa equivale a 2π rad ≈ 6,28318 rad.

Fórmula de Conversión:

α° / 180° = R rad / π rad

Sexagesimal (°) 0° 30° 45° 60° 90° 180° 270° 360°
Radianes (rad) 0 π/6 π/4 π/3 π/2 π 3π/2 2π

5.2 Circunferencia Trigonométrica Unitaria

Definición de Razones Trigonométricas

Una circunferencia trigonométrica es una circunferencia centrada en el origen (0,0) de radio R = 1 (ecuación x2 + y2 = 1).
Dado un ángulo α, el rayo terminal intercepta a la circunferencia en el punto P(x, y):

sen(α) = y        cos(α) = x        tg(α) = y / x   (con x ≠ 0)

cosec(α) = 1 / y        sec(α) = 1 / x        cotg(α) = x / y

Circunferencia Trigonométrica Unitaria (R=1): P(x,y) = (cosα, senα)

Identidad Pitagórica Fundamental

Puesto que P(x, y) pertenece a la circunferencia unitaria x2 + y2 = 1, sustituyendo x = cos α e y = sen α obtenemos:

sen2(α) + cos2(α) = 1

Dividiendo por cos2(α) y sen2(α) surgen las relaciones derivadas:
• 1 + tg2(α) = sec2(α)
• 1 + cotg2(α) = cosec2(α)

5.3 Funciones Trigonométricas Reales

Función Seno: y = sen(x)

• Dominio: Dom = R
• Imagen: Im = [−1; 1]
• Período: T = 2π
• Paridad: Impar (sen(−x) = −sen(x))
• Ceros: x = kπ (con k ∈ Z).

Función Coseno: y = cos(x)

• Dominio: Dom = R
• Imagen: Im = [−1; 1]
• Período: T = 2π
• Paridad: Par (cos(−x) = cos(x))
• Ceros: x = π/2 + kπ (con k ∈ Z).

Función Tangente: y = tg(x)

• Dominio: Dom = R ∖ {π/2 + kπ / k ∈ Z}
• Imagen: Im = R
• Período: T = π
• Asíntotas Verticales: Rectas x = π/2 + kπ.

Ondas Trigonométricas Periódicas: y = sen(x) (azul) y y = cos(x) (rojo)

5.4 Relaciones Trigonométricas Inversas

Dado que las funciones trigonométricas son periódicas y no inyectivas en R, para definir sus funciones inversas es indispensable restringir sus dominios.

Funciones Inversas Principales

• Arcoseno: y = arcsen(x) ⇔ sen(y) = x
    Dominio: [−1; 1]   ⇒   Imagen: [−π/2; π/2].

• Arcocoseno: y = arccos(x) ⇔ cos(y) = x
    Dominio: [−1; 1]   ⇒   Imagen: [0; π].

• Arcotangente: y = arctg(x) ⇔ tg(y) = x
    Dominio: R   ⇒   Imagen: (−π/2; π/2) (con asíntotas horizontales en y = ±π/2).

5.5 Ecuaciones Trigonométricas

Resolver una ecuación trigonométrica consiste en determinar todos los ángulos x ∈ R que satisfacen la igualdad planteada.

Ejemplo 1: 2 · sen(x) − 1 = 0

1. Despejamos la función básica:  sen(x) = 1/2.
2. Hallamos las soluciones en el primer ciclo [0; 2π):
    • Primer cuadrante: x1 = arcsen(1/2) = π/6 (30°).
    • Segundo cuadrante (seno positivo): x2 = π − π/6 = 5π/6 (150°).
3. Expresamos la Solución General en R sumando múltiplos enteros del período 2kπ:

S = { π/6 + 2kπ  ;  5π/6 + 2kπ  /  k ∈ Z }

Ejemplo 2: cos2(x) − sen2(x) = 0

1. Usamos la identidad del coseno del ángulo doble:  cos(2x) = cos2(x) − sen2(x).
2. La ecuación resulta:  cos(2x) = 0.
3. El coseno se anula en los múltiplos impares de π/2:
    2x = π/2 + kπ  ⇒  x = π/4 + k · π/2   (con k ∈ Z).
Conjunto Solución:  S = { π/4 + k · π/2  /  k ∈ Z }.

5.6 Ejercicios Prácticos del Módulo

Ejercicio 208: Sistema de Ángulos

La suma de 2 ángulos es 1,93295 rad y su diferencia es 10°45'. Hallar la medida de cada uno.

Ver Resolución

1. Convertimos la suma a sexagesimal:
    S = (1,93295 · 180°) / π ≈ 110°45'.
2. Planteamos el sistema lineal:
    α + β = 110°45'
    α − β = 10°45'
3. Sumamos ambas ecuaciones: 2α = 121°30' ⇒ α = 60°45'.
4. Restamos ambas ecuaciones: 2β = 100° ⇒ β = 50°.

Ejercicio Ecuación: tg2(x) − 3 = 0

Resolver la ecuación trigonométrica en [0; 2π).

Ver Resolución

1. Despejamos:  tg2(x) = 3 ⇒ tg(x) = ±√3.
2. Ángulo de referencia: arctg(√3) = π/3 (60°).
3. Cuatro soluciones en los 4 cuadrantes:
    • I Cuadrante: x1 = π/3
    • II Cuadrante: x2 = π − π/3 = 2π/3
    • III Cuadrante: x3 = π + π/3 = 4π/3
    • IV Cuadrante: x4 = 2π − π/3 = 5π/3
Solución: S = {π/3, 2π/3, 4π/3, 5π/3}.
