2.1 La Función Logarítmica

y = loga(x)

La función exponencial elemental f: R → R+ / f(x) = ax (con a > 0 y a ≠ 1) es una función biyectiva, por lo cual admite función inversa.

Definición Formal como Función Inversa

Se define la función logarítmica de base a como la función inversa de la exponencial de base a:

f−1: R+ → R  /  f−1(x) = loga(x)

Por la definición de función inversa, se cumple la equivalencia fundamental:

y = loga(x)   ⇔   ay = x     (con x > 0,   a > 0,   a ≠ 1)

Dominio e Imagen:
• Dominio: Dom(loga x) = R+ = (0; ∞).
• Imagen: Im(loga x) = R.
• Asíntota Vertical: Posee una asíntota vertical en la recta x = 0 (el eje y).

¡ADVERTENCIA CRUCIAL!

Dado que Dom(loga x) = (0; ∞), no existen en R los logaritmos de números negativos ni el logaritmo de cero.

2.1.1 Logaritmos Decimal y Natural

Logaritmo Decimal (Base 10)

Si la base es a = 10, se omite el subíndice y se escribe simplemente:

log10(x) = log(x)

Es el logaritmo estándar de las calculadoras científicas y de las escalas logarítmicas como el pH y los decibelios.

Logaritmo Natural o Neperiano (Base e)

Si la base es el número irracional a = e ≈ 2,71828..., se denota por:

loge(x) = ln(x)

Es la función inversa de la exponencial natural y = ex y es fundamental en el cálculo infinitesimal.

2.2 Propiedades de los Logaritmos

Para todo a, b > 0 (con a, b ≠ 1) y para cualesquiera argumentos positivos x, z > 0, se verifican las siguientes propiedades algebraicas:

Leyes Operatorias Fundamentales

  • • Logaritmo de 1:  loga(1) = 0   (porque a0 = 1).
  • • Logaritmo de la base:  loga(a) = 1   (porque a1 = a).
  • • Logaritmo de un Producto:  loga(x · z) = loga(x) + loga(z)
  • • Logaritmo de un Cociente:  loga(x / z) = loga(x) − loga(z)
  • • Logaritmo de una Potencia:  loga(xn) = n · loga(x)  (para todo n ∈ R).
  • • Fórmula de Cambio de Base:  loga(x) = logb(x) / logb(a)
  • • Inyectividad del Logaritmo:  loga(x) = loga(z)  ⇔  x = z.

2.4 Ejemplos Analíticos Desarrollados

Ejemplo 1: Cálculo por Definición de Logaritmos

a) Resolver  log3(x) = 2:
Aplicando la definición y = loga(x) ⇔ ay = x:
32 = x  ⇒  x = 9.

b) Resolver  log3(1/27) = y:
Aplicando definición: 3y = 1/27  ⇒  3y = 3−3  ⇒  y = −3.

Ejemplo 2: Análisis Comparativo Completo de f(x) = log3(x) y g(x) = log1/3(x)

Propiedad f(x) = log3(x) g(x) = log1/3(x)
Dominio (0; ∞) = R+ (0; ∞) = R+
Imagen R R
Raíz (y=0) (1, 0) (1, 0)
Ordenada (x=0) No tiene (0 ∉ Dom) No tiene (0 ∉ Dom)
Monotonía Estrictamente Creciente Estrictamente Decreciente
Asíntota Vert. x = 0 (eje y) x = 0 (eje y)

Ejemplo 3: Análisis de Dominio y Asíntotas en Logaritmos Desplazados

Dada la función  h(x) = ln(2x − 6):

1. Cálculo del Dominio:
Exigimos que el argumento sea estrictamente mayor que cero:
    2x − 6 > 0  ⇒  2x > 6  ⇒  x > 3.
Por lo tanto:  Dom h = (3; ∞).

2. Asíntota Vertical:
El valor que anula el argumento genera la asíntota vertical:
    2x − 6 = 0  ⇒  x = 3 es la Asíntota Vertical.

3. Raíz (h(x) = 0):
    ln(2x − 6) = 0  ⇒  2x − 6 = e0  ⇒  2x − 6 = 1  ⇒  2x = 7  ⇒  x = 7/2 = 3.5.

Ejercicio 180: Completamiento Teórico de Logaritmos

Completar teniendo en cuenta la función f(x) = loga(x):

Ver Respuestas

a) Todas las curvas cortan al eje x en el punto: (1, 0).

b) Si a > 1, entonces f es una función: Creciente.

c) Si 0 < a < 1, entonces f es una función: Decreciente.

d) Las gráficas de y = loga x e y = log1/a x son simétricas con respecto al: eje x.

Ejercicio 181: Función Inversa de g(x) = ex

Dada la función exponencial natural g(x) = ex, obtener por simetría la fórmula funcional de su inversa g−1(x).

Ver Resolución

1. Escribimos y = ex.
2. Intercambiamos variables: x = ey.
3. Aplicamos definición de logaritmo: y = loge(x) = ln(x).

Por lo tanto: g−1(x) = ln(x)  con Dom = (0; ∞) e Im = R.
