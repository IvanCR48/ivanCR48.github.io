1.1 Definición

f(x) = k · ax

Llamamos función exponencial a toda función real donde la variable independiente se encuentra en el exponente.

Definición Formal

Una función exponencial es una función de la forma:

f(x) = k · ax

Donde:
• a es la base, una constante real positiva distinta de 1:  a > 0  y  a ≠ 1.
• k es el coeficiente principal, una constante real no nula:  k ∈ R ∖ {0}.
• x es la variable independiente (exponente).

Dominio: El dominio de toda función exponencial elemental es el conjunto de los números reales:

Dom f = R

1.2 Función Potencial vs. Exponencial y Exponencial Natural

Distinción Conceptual Clave

Es muy importante no confundir ambos tipos de funciones:

  • • Función Potencial: y = xn (base variable, exponente constante). Ejemplo: y = x2.
  • • Función Exponencial: y = ax (base constante, exponente variable). Ejemplo: y = 2x.
La Función Exponencial Natural

Existe una función exponencial de notable importancia donde la base es el número irracional e ≈ 2,7182818... (número de Euler):

f(x) = ex

Se utiliza ampliamente como modelo matemático para describir el crecimiento poblacional, desintegración radiactiva, enfriamiento y procesos económicos.

1.3 Análisis de Parámetros y Simetrías

Analicemos cómo varían las gráficas de la función f(x) = k · ax según los valores de la base a y del coeficiente k.

Caso Base: f(x) = ax (con k = 1)

Características Comunes:
• Ordenada al origen: Cortan al eje de ordenadas en el punto (0, 1) porque a0 = 1.
• Raíces: No cortan al eje de abscisas (ax > 0 para todo x).
• Imagen: Im f = R+ = (0; ∞).
• Asíntota Horizontal: Tienen como asíntota al eje x (recta y = 0).

Diferencias según la Base a:
• Si a > 1: La función es estrictamente creciente.
• Si 0 < a < 1: La función es estrictamente decreciente.
• Simetría Especular: Las curvas de bases recíprocas (ej: y = 2x e y = (1/2)x = 2−x) son simétricas respecto al eje y.

Simetría respecto al eje y: f(x) = 2x (azul) y g(x) = (1/2)x (violeta)

Efecto del Coeficiente k: f(x) = k · ax

• Ordenada al origen: Corta al eje y en el punto (0, k).
• Si k > 0: La gráfica está sobre el eje x y su imagen es Im f = R+ = (0; ∞).
• Si k < 0: La gráfica se invierte verticalmente y su imagen es Im f = R− = (−∞; 0).
• Simetría Especular respecto al eje x: Las curvas con coeficientes opuestos (k y −k) son simétricas con respecto al eje x.

1.4 Desplazamientos y Asíntotas Horizontales

Al sumar o restar una constante a la función exponencial, la curva sufre un desplazamiento vertical que modifica la posición de su asíntota horizontal.

Forma General Transformada

f(x) = k · ax − c + d

• Asíntota Horizontal (A.H.): Viene dada por el término independiente y = d.
• Desplazamiento Vertical (d): Mueve la curva d unidades hacia arriba (si d > 0) o hacia abajo (si d < 0).
• Conjunto Imagen: Depende de d y del signo de k:
    • Si k > 0 ⇒ Im f = (d; ∞).
    • Si k < 0 ⇒ Im f = (−∞; d).
• Cálculo de Raíces (Eje x): Para hallar la intersección con el eje x se resuelve la ecuación:
    k · ax − c + d = 0 ⇒ ax − c = −d / k.
    Tiene raíz real sólo si (−d / k) > 0.

Desplazamientos Verticales: f(x) = 3x (azul), g(x) = 3x + 1 (verde), h(x) = 3x − 1 (rojo)

Ejemplo 2: Análisis de Coeficiente Negativo

Sean  f(x) = −2x + 4  y  g(x) = −2x − 1.

1. Para f(x) = −2x + 4:
• Dominio: Dom f = R.
• Asíntota Horizontal: y = 4.
• Imagen: Debido al signo negativo del coeficiente (k = −1), la curva decrece desde la asíntota hacia −∞ ⇒ Im f = (−∞; 4).
• Ordenada al origen: f(0) = −20 + 4 = −1 + 4 = 3 ⇒ punto (0, 3).
• Cálculo de Raíz (f(x) = 0):
    −2x + 4 = 0 ⇒ 2x = 4 ⇒ 2x = 22 ⇒ x = 2.

2. Para g(x) = −2x − 1:
• Dominio: Dom g = R.
• Asíntota Horizontal: y = −1.
• Imagen: Im g = (−∞; −1).
• Ordenada al origen: g(0) = −20 − 1 = −1 − 1 = −2 ⇒ punto (0, −2).
• Cálculo de Raíz (g(x) = 0):
    −2x − 1 = 0 ⇒ 2x = −1.
    Como 2x > 0 para todo x real, esta ecuación no tiene solución en R ⇒ g(x) no tiene raíces.

1.6 Ejercicios Prácticos del Módulo

Ejercicio 171: Puntos sobre y = 4x

Dada la función y = 4x, indicar V o F y justificar:

Ver Justificaciones

b_1) El punto (1, 0) pertenece a la gráfica:
Falso. Para x = 1, y = 41 = 4 ≠ 0. El punto que pertenece es (1, 4).

b_2) El valor de y para (-2, y) es 1/16:
Verdadero. Para x = -2, y = 4-2 = 1 / 42 = 1/16.

b_3) El punto (-1, -4) pertenece a la gráfica:
Falso. Para x = -1, y = 4-1 = 1/4 ≠ -4. Dado que ax > 0, ningún punto con ordenada negativa pertenece a y = 4x.

b_4) El valor de x para que (x, 2) pertenezca a la gráfica es 1/2:
Verdadero. Resolvemos 4x = 2 ⇒ (22)x = 21 ⇒ 22x = 21 ⇒ 2x = 1 ⇒ x = 1/2.

Ejercicio 172: Simetría de y = (1/3)x

Hallar la fórmula de la función exponencial simétrica a y = (1/3)x respecto al eje y (x = 0).

Ver Resolución

Para obtener la simétrica respecto al eje y, reemplazamos x por −x:
y = (1/3)−x = [(3)−1]−x = 3x.
Por lo tanto, la función simétrica es y = 3x (base recíproca).

Ejercicio 173: Proposiciones Teóricas

Analizar la veracidad de las siguientes afirmaciones sobre f(x) = ax:

Ver Justificaciones

a) Si a > 0 y a ≠ 1, entonces Dom f = R e Im f = R+:
Verdadero. El exponente puede tomar cualquier valor real y la potencia de base positiva siempre es estrictamente positiva.

b) Si 0 < a < 1, la función f(x) = ax es creciente:
Falso. Cuando la base está entre 0 y 1, a mayor exponente la potencia decrece, por lo que la función es decreciente.

c) Existe algún x real tal que f(x) = 0 para f(x) = ax:
Falso. La función exponencial elemental posee la recta y = 0 como asíntota horizontal y nunca alcanza el valor cero (ax > 0 para todo x).
