Breve historia del numero

La nocion de numero es tan antigua como el hombre mismo. Las tribus mas primitivas disponian de simbolos para distinguir entre uno, dos, tres...

Prehistoria

El numero natural

El hombre primitivo poseia la idea del numero natural (1, 2, 3, 4...). Hasta la Edad de Bronce no aparecen sistemas de numeracion potentes.

~2000 a.C. — Egipto

El numero fraccionario

Los egipcios comienzan a manejar fracciones sencillas: 1/2, 1/3, 2/3, 6/7... Los griegos elevan estas manipulaciones a la categoria de Ciencia.

Siglo V a.C. — Grecia

Aparicion del irracional

Los pitagoricos descubren que la diagonal y el lado de un cuadrado tienen un cociente que no es una fraccion. Llamaron a este tipo de numero alogos (irracional).

Siglo XVI-XVII — Europa

El numero negativo

La resolucion de ecuaciones algebraicas conduce a los numeros negativos. En geometria: lo negativo indica retroceso, lo positivo avance. Los chinos ya los usaban desde el siglo IV a.C. con bolas de abaco.

Siglo XIX

Los complejos

Al resolver ecuaciones como x² - 2x + 26 = 0 aparecen expresiones como √(-25). Su fundamentacion moderna no llego hasta mediados del siglo XIX.

Numeros Naturales

N

Definicion

Son los numeros que utilizamos para contar:

N = { 1, 2, 3, 4, 5, 6, 7, ... }

Si incluimos el cero, se escribe N0 = { 0, 1, 2, 3, 4, ... }

Recta numerica de los Naturales

Cerrado bajo la suma

a + b ∈ N

3 + 5 = 8 ∈ N

Cerrado bajo el producto

a · b ∈ N

4 · 7 = 28 ∈ N

NO cerrado bajo la resta

a - b ∉ N (si a < b)

3 - 9 = -6 ∉ N

NO cerrado bajo la division

a / b ∉ N (en general)

7 / 2 = 3.5 ∉ N

Problema motivador

La ecuacion x + 9 = 3 tiene coeficientes naturales, pero no tiene solucion en N. Para resolverla necesitamos restar 9 en ambos miembros:

x + 9 - 9 = 3 - 9
x = -6

Necesitamos un conjunto mas grande: los numeros enteros.

Numeros Enteros

Z

Definicion

Son los numeros naturales, sus opuestos (negativos) y el cero:

Z = { ..., -3, -2, -1, 0, 1, 2, 3, ... }

La letra Z proviene del aleman Zahlen (numeros).

Recta numerica de los Enteros

Relacion de contenencia

N ⊂ Z

Todo numero natural es tambien un numero entero.

Cerrado bajo la suma

a + b ∈ Z

(-3) + 5 = 2 ∈ Z

Cerrado bajo la resta

a - b ∈ Z

2 - 7 = -5 ∈ Z

Cerrado bajo el producto

a · b ∈ Z

(-3) · 4 = -12 ∈ Z

NO cerrado bajo la division

a / b ∉ Z (en general)

5 / 3 ∉ Z

Problema motivador

La ecuacion 2x + 5 = 6 no tiene solucion entera:

2x = 6 - 5 = 1
x = 1/2

Necesitamos un conjunto mas grande: los numeros racionales.

Numeros Racionales

Q

Definicion

La palabra "racional" viene de razon o proporcion. Un numero racional es el resultado de la razon entre dos enteros:

Q = { p/q | p ∈ Z, q ∈ Z, q ≠ 0 }

Donde p es el numerador y q el denominador.

Ejemplos y su representacion decimal

Fraccion Decimal Tipo de decimal
1/2 0,5 Exacto (finito)
-3/5 -0,6 Exacto (finito)
2/4 0,5 Exacto (igual a 1/2)
22/3 7,333... Periodico (periodo: 3)
1821/7 260,142857... Periodico (periodo: 142857)
13/6 2,1666... Semiperiodico (periodo: 6)

Recta numerica: los racionales llenan los "huecos"

Propiedad clave de los racionales

Todo numero racional tiene una expansion decimal que es:

Decimal exacto (finito)

1/4 = 0,25

Cantidad finita de digitos despues de la coma.

Decimal periodico

22/3 = 7,333...

Un grupo finito de digitos (el periodo) se repite infinitamente.

Decimal semiperiodico

13/6 = 2,1666...

Comienza con digitos no repetitivos (anteperiodo) y luego se repite un periodo.

Nota importante

Distintas fracciones pueden representar el mismo numero racional. Por ejemplo: 1/2 = 2/4 = 3/6 = 50/100. Se llaman fracciones equivalentes.

Problema motivador

La ecuacion x² - 2 = 0 no tiene solucion racional:

x² = 2 ⇒ x = √2 = 1,41421356...

Este decimal es infinito y no tiene periodo. No se puede expresar como fraccion p/q. Necesitamos los numeros irracionales.

Numeros Irracionales

I

Definicion

Son aquellos cuya expresion decimal es infinita y no tiene periodo. No pueden expresarse como fraccion p/q.

I = { x ∈ R | x ∉ Q }

Irracionales famosos

Simbolo Nombre Valor aproximado Descripcion
π Pi 3,14159265358... Razon entre la circunferencia y su diametro
e Numero de Euler 2,71828182845... Base del logaritmo natural
√2 Raiz de 2 1,41421356237... Diagonal de un cuadrado de lado 1
√3 Raiz de 3 1,73205080756... Diagonal de un rectangulo 1x√2
√p Raiz de primo --- √p es irracional para todo primo p

Como construir un irracional

Podemos generar un irracional no famoso construyendo un decimal infinito sin periodo:

1 1,1
2 1,101
3 1,101001
4 1,1010010001
... 1,101001000100001...

Cada vez agregamos un "1" separado por una cantidad creciente de ceros. No hay forma de encontrar un grupo de digitos que se repita, por lo tanto es irracional.

El cuadrado de Pitagoras: origen de √2

Racional vs Irracional: la diferencia clave

Racional (Q)

22/7 = 3,142857142857...

El bloque 142857 se repite para siempre.

Se puede escribir como fraccion p/q

Irracional (I)

π = 3,14159265358979323846...

Los digitos continuan sin repetir ningun patron.

NO se puede escribir como fraccion p/q

Numeros Reales

R

Definicion

La union de los numeros racionales y los irracionales forman el conjunto de los numeros reales:

R = Q ∪ I

Cada punto de la recta numerica corresponde a exactamente un numero real, y cada numero real corresponde a un punto de la recta.

Diagrama de conjuntos anidados

Cadena de contenencia

N ⊂ Z ⊂ Q ⊂ R

Todo natural es entero, todo entero es racional, y todo racional es real.
Los irracionales son reales que no son racionales: I = R ∖ Q.

Limite del campo real

Los reales tampoco resuelven todas las ecuaciones. La ecuacion x² + 1 = 0 no tiene solucion en R, ya que requiere x = √(-1). Para eso existen los numeros complejos (tema posterior).

Propiedades de los Numeros Reales

Sean a, b, c numeros reales.

Propiedad Suma (+) Producto (·)
Conmutativa a + b = b + a a · b = b · a
Asociativa (a + b) + c = a + (b + c) (a · b) · c = a · (b · c)
Elemento neutro a + 0 = a a · 1 = a
Elemento inverso a + (-a) = 0 a · (1/a) = 1  (a ≠ 0)

Distributividad

Valida

a · (b + c) = a·b + a·c

El producto distribuye sobre la suma.

Valida

(a + b) / c = a/c + b/c

El cociente distribuye sobre la suma si la suma esta en el numerador (c ≠ 0).

NO valida

a / (b + c) ≠ a/b + a/c

La suma en el denominador no se puede distribuir.

Ejemplo: 12/(3+2) = 12/5 = 2,4   pero   12/3 + 12/2 = 4 + 6 = 10

Propiedades sobre igualdades

Si a = b, entonces a + c = b + c

Se puede sumar (o restar) lo mismo a ambos miembros.

Si a = b, entonces a · c = b · c

Se puede multiplicar ambos miembros por lo mismo.

Si a = b y c ≠ 0, entonces a/c = b/c

Se puede dividir ambos miembros por lo mismo (no nulo).

Si a · b = 0, entonces a = 0 o b = 0

Propiedad del producto nulo: si un producto es cero, al menos uno de los factores es cero.
