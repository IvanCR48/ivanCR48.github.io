Desigualdades en los Reales

<

El conjunto de los numeros reales R es un conjunto ordenado. Podemos describir este orden de dos maneras complementarias:

Descripcion Geometrica

a < b

Significa que el punto a esta a la izquierda del punto b en la recta numerica.

Descripcion Algebraica

a < b  ⇒  b - a > 0

Significa que la diferencia b - a es un numero estrictamente positivo.

Desigualdades Estrictas

<   (menor que)   o   >   (mayor que)

Ejemplo: 3 < π   ya que π - 3 ≈ 0.14159 es positivo.

Desigualdades Debiles

≤   (menor o igual)   o   ≥   (mayor o igual)

Ejemplo: 6 ≤ 6   (es verdadera ya que 6 = 6).

Propiedad de Tricotomia

Para cualesquiera numeros reales a y b, se cumple una y solo una de las siguientes afirmaciones:

a < b     o     a = b     o     a > b

Propiedades fundamentales

Sean a, b, c numeros reales:

Propiedad Condicion Efecto algebraico Ejemplo
Transitividad Si a < b y b < c a < c Si 2 < 5 y 5 < 9 ⇒ 2 < 9
Suma / Resta Cualquier c ∈ R a + c < b + c Si 3 < 7 ⇒ 3 + 2 < 7 + 2 ⇒ 5 < 9
Producto Positivo c > 0 (positivo) a · c < b · c Si 2 < 3 ⇒ 2(4) < 3(4) ⇒ 8 < 12
Producto Negativo c < 0 (negativo) a · c > b · c Si 2 < 3 ⇒ 2(-4) > 3(-4) ⇒ -8 > -12

Importante: Multiplicacion por numeros negativos

Al multiplicar o dividir ambos miembros de una desigualdad por un numero negativo, el sentido de la desigualdad se invierte.

Intervalos reales

Los intervalos son subconjuntos de R descritos por desigualdades. Usamos corchetes [ ] para incluir el extremo, y parentesis ( ) para excluirlo.

Intervalos Acotados (con dos extremos)

Notacion Tipo Definicion de conjunto Representacion Grafica
[a; b] Cerrado { x ∈ R / a ≤ x ≤ b }
(a; b) Abierto { x ∈ R / a < x < b }
[a; b) Semiabierto (izq) { x ∈ R / a ≤ x < b }
(a; b] Semiabierto (der) { x ∈ R / a < x ≤ b }

Intervalos No Acotados (semirrectas)

Usamos el simbolo de infinito ∞ o -∞ para indicar que el intervalo continua por siempre. Siempre se escribe parentesis junto a ∞.

Notacion Definicion de conjunto Representacion Grafica
[a; ∞) { x ∈ R / x ≥ a }
(a; ∞) { x ∈ R / x > a }
(-∞; a] { x ∈ R / x ≤ a }
(-∞; a) { x ∈ R / x < a }

Operaciones entre conjuntos e intervalos

Dado que los intervalos son conjuntos, podemos aplicarles las operaciones de conjuntos:

Union (A ∪ B)

{ x / x ∈ A  o  x ∈ B }

Elementos que pertenecen a cualquiera de los dos conjuntos.

Interseccion (A ∩ B)

{ x / x ∈ A  y  x ∈ B }

Elementos comunes a ambos conjuntos.

Diferencia (A - B)

{ x / x ∈ A  y  x ∉ B }

Elementos de A que NO pertenecen a B.

Complemento (A')

{ x / x ∈ U  y  x ∉ A }

Elementos que no estan en A (con U = R).

Caso de estudio practico (PDF)

Sean:
A = { x ∈ R / x < 0 } = (-∞; 0)
B = { x ∈ R / -3 ≤ x < 2 } = [-3; 2)

Operacion Desarrollo en Intervalos Representacion en la recta
A ∪ B (-∞; 2)
A ∩ B [-3; 0)
A - B (-∞; -3)
B - A [0; 2)

Inecuaciones Lineales

Una inecuacion es una desigualdad condicional que involucra una o mas variables. Resolverla significa encontrar el conjunto de valores que la hacen verdadera.

Resolucion de Inecuacion Lineal (Ejemplo 1)

x/2 + 1 < 15 + 3x

1 Restamos 1 a ambos miembros: x/2 < 14 + 3x
2 Multiplicamos por 2 ambos miembros (positivo, no cambia el sentido): x < 2(14 + 3x)
3 Distribuimos: x < 28 + 6x
4 Restamos 6x en ambos miembros: -5x < 28
5 Dividimos por -5: Invertimos el sentido! ⇒ x > -28/5

La solucion expresada como intervalo es: (-28/5; ∞).

Inecuaciones con Valor Absoluto

|x|≤a

Reglas generales de equivalencia (con a > 0)

Caso A: Modulo menor o igual

|x| ≤ a

Es equivalente a una doble desigualdad:

-a ≤ x ≤ a

Representa los puntos cuya distancia al origen es menor o igual a a.

Caso B: Modulo mayor o igual

|x| ≥ a

Es equivalente a la union de dos semirrectas:

x ≤ -a    o    x ≥ a

Representa los puntos cuya distancia al origen es mayor o igual a a.

Ejemplo A: |x| ≤ 3

|x| ≤ 3  ⇒  -3 ≤ x ≤ 3

El conjunto solucion es el intervalo cerrado: [-3; 3].

Ejemplo B: |x| ≥ 3

|x| ≥ 3  ⇒  x ≤ -3    o    x ≥ 3

El conjunto solucion es la union de dos intervalos: (-∞; -3] ∪ [3; ∞).

Ejemplo C: |x - 2| < 3

-3 < x - 2 < 3
-3 + 2 < x < 3 + 2 ⇒ -1 < x < 5

Sumamos 2 a todos los miembros de la desigualdad. El intervalo solucion es: (-1; 5).

Ejemplo D: |x - 4| ≥ 2

x - 4 ≤ -2     o     x - 4 ≥ 2
x ≤ 2     o     x ≥ 6

La distancia entre x y 4 es mayor o igual a 2. La solucion es la union de dos intervalos: (-∞; 2] ∪ [6; ∞).

Precaucion: Separar en las dos ramas correctas

Es muy comun cometer el error de plantear una inecuacion como |x - 4| ≥ 2 omitiendo la rama negativa, lo que provoca la perdida de la mitad de las soluciones. Recuerda siempre separar la desigualdad con ≥ en sus dos partes disjuntas:

(rama negativa) x - 4 ≤ -2     y     (rama positiva) x - 4 ≥ 2.
