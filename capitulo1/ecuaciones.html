Igualdad, Identidad y Ecuacion

=

El uso del signo "igual" es fundamental en matematica. Sin embargo, segun como se plantee, estamos expresando cosas completamente diferentes:

Identidad

(x - 5)(x + 5) = x2 - 25

Es una afirmacion. La igualdad es verdadera para cualquier valor real que tome la variable x.

Siempre verdadera

Ecuacion

x2 - 25 = 0

Es una pregunta. Sola es verdadera para determinados valores específicos de la variable x (en este caso, x = 5 y x = -5).

Condicional

Axiomas de Euclides (Elementos, ~300 a.C.)

1. Dos cosas iguales a una tercera cosa, son iguales entre si.
2. Si cantidades iguales se suman a numeros iguales, los totales resultan ser iguales.
3. Si cantidades iguales se restan de numeros iguales, los residuos son iguales.
4. Cosas que coinciden con otra, son iguales entre si.

Reglas para modificar ecuaciones

Para resolver una ecuacion, la modificamos paso a paso hasta dejar la variable sola. Para asegurar que no alteramos las soluciones originales, aplicamos las siguientes reglas equivalentes:

Regla 1: Suma / Resta

Si a = b ⇒ a + c = b + c

Se puede sumar o restar la misma cantidad a ambos miembros sin alterar la solucion.

Regla 2: Multiplicacion / Division

Si a = b ⇒ a · c = b · c   (c ≠ 0)

Se puede multiplicar o dividir ambos miembros por una cantidad no nula sin alterar la solucion.

Ecuaciones Lineales

ax+b

Definicion

Una ecuacion de primer grado o ecuacion lineal es aquella de la forma:

ax + b = 0    (a ≠ 0)

Donde la variable x aparece elevada unicamente a la primera potencia. Su solucion unica es:

x = -b / a

Resolucion paso a paso (Ejemplo 1)

12x - 9 = 5x + 5

1 Sumamos 9 a ambos miembros: 12x = 5x + 14
2 Restamos 5x a ambos miembros: 7x = 14
3 Dividimos ambos miembros por 7: x = 2

Verificacion de la solucion

Reemplazamos x = 2 en la ecuacion original para verificar que la igualdad sea verdadera:

12(2) - 9 = 5(2) + 5
24 - 9 = 10 + 5
15 = 15    (Afirmacion Verdadera)

Ecuaciones Racionales transformables

Muchas ecuaciones con la variable en el denominador se pueden resolver transformandolas en lineales, siempre teniendo cuidado de excluir los valores que anulan los denominadores (restricciones de dominio).

Ejemplo 2 (Con Solucion)

1 Establecemos restricciones: 2x - 2 ≠ 0 ⇒ x ≠ 1;    x + 1 ≠ 0 ⇒ x ≠ -1
2 Multiplicamos cruzado (o por el comun denominador): 3(x + 1) = 2(2x - 2)
3 Distribuimos: 3x + 3 = 4x - 4
4 Despejamos x: 3 + 4 = 4x - 3x ⇒ x = 7

Verificando en la original: 3 / (2(7) - 2) = 3/12 = 1/4;    2 / (7 + 1) = 2/8 = 1/4. Como x = 7 no viola ninguna restriccion, x = 7 es la solucion.

Ejemplo 3 (Sin Solucion / Solucion Extranea)

1 Restriccion inicial: x - 3 ≠ 0 ⇒ x ≠ 3
2 Multiplicamos toda la ecuacion por (x - 3): 3x + 1(x - 3) = 9
3 Simplificamos: 3x + x - 3 = 9 ⇒ 4x - 3 = 9
4 Despejamos x: 4x = 12 ⇒ x = 3

¡Cuidado! Obtuvimos x = 3, pero teniamos la restriccion x ≠ 3 (ya que provocaria division por cero). Por lo tanto, la solucion es extranea y concluimos que la ecuacion no tiene solucion.

Tipo Subtipo Cantidad de Soluciones Ejemplo algebraico

Indeterminada (Identidades) Infinitas soluciones 2(x + 1) = 2x + 2 ⇒ 2x + 2 = 2x + 2

Ecuaciones lineales con parametros (k)

Consiste en analizar para que valores del parametro k la ecuacion es compatible o incompatible.

Caso de estudio: -2kx + 1 = x + 2k

Agrupamos las x de un lado y las constantes del otro:

-2kx - x = 2k - 1
x(-2k - 1) = 2k - 1
x(2k + 1) = 1 - 2k

Ahora analizamos el coeficiente de x, que es (2k + 1):

Si 2k + 1 ≠ 0 (k ≠ -1/2)

Compatible Determinada

Tiene solucion unica: x = (1 - 2k)/(2k + 1)

Si 2k + 1 = 0 (k = -1/2)

Incompatible

Queda x(0) = 1 - 2(-1/2) ⇒ 0 = 2 (Absurdo, no tiene solucion)

Problemas de aplicacion (Palabras a ecuaciones)

v·t

Ejemplo A: El dinero de Juan

Juan dice: "Si tuviera $12 mas de lo que tengo y despues duplicara esa cantidad, tendria $60 mas de lo que tengo".

Frase textual Traduccion algebraica
El dinero que tiene Juan x
$12 mas de lo que tiene x + 12
El doble de esa cantidad 2(x + 12)
$60 mas de lo que tiene x + 60

2(x + 12) = x + 60
2x + 24 = x + 60 ⇒ x = 36

Conclusion: Juan tiene $36.

Ejemplo B: Problema de Velocidad y Distancia

A las 2:00 pm Don Lenton viaja al este a 45 km/h. Una hora mas tarde (3:00 pm), Don Veloz sale detras de el a 60 km/h. ¿A que hora lo alcanzara?

Esquema del movimiento

Analisis del tiempo

Sea t el tiempo (en horas) desde las 2:00 pm.
Don Lenton maneja t horas.
Don Veloz sale 1 hora despues, asi que maneja t - 1 horas.

Analisis de distancia

La distancia recorrida por ambos al encontrarse es la misma.
d = v · t
Don Lenton: 45t
Don Veloz: 60(t - 1)

45t = 60(t - 1)
45t = 60t - 60 ⇒ 60 = 15t ⇒ t = 4

Conclusion: Se encuentran 4 horas despues de las 2:00 pm, es decir, a las 6:00 pm.
