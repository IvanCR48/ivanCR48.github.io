5. Funcion Racional

f(x) = P(x) / Q(x)

Llamamos funcion racional al cociente de dos polinomios reales. Su dominio esta restringido por los valores que anulan al denominador.

Definicion Matematica

Una funcion f: A → R es racional si es de la forma:

f(x) = P(x) / Q(x)

Donde P(x) y Q(x) son polinomios con Q(x) ≠ 0.
El dominio de definicion es:

Dom f = { x ∈ R  |  Q(x) ≠ 0 }

Analisis de Asintotas

Las asintotas son rectas a las cuales la grafica de la funcion se aproxima indefinidamente a medida que la variable tiende a un determinado valor o al infinito.

Asintota Vertical (A.V.)

x = a

Si a es una raiz del denominador Q(x) y no anula al numerador P(x) (P(a) ≠ 0), entonces la recta vertical x = a es una A.V.

Asintota Horizontal (A.H.)

y = k

• Si gr P < gr Q ⇒ A.H. es y = 0.
• Si gr P = gr Q ⇒ A.H. es y = an / bm (cociente de coeficientes principales).
• Si gr P > gr Q ⇒ No tiene A.H.

Grafica de la funcion racional elemental y = 1/x

Ceros de la funcion y Puntos Vacios ("Agujeros")

• Ceros: Son las raices del numerador P(x) que pertenecen al Dom f.
• Puntos vacios: Si un valor x = a anula simultaneamente a P(x) y Q(x), al simplificar la fraccion el factor (x - a) se elimina. La grafica sera identica a la simplificada, pero con un punto vacio o agujero en x = a.

Ejemplo (PDF):  g(x) = (x2 - 9) / (x - 3).
Dominio: Dom g = R ∖ {3}.
Factorizando:  g(x) = [(x - 3)(x + 3)] / (x - 3) = x + 3  (si x ≠ 3).
La grafica es la recta  y = x + 3  con un "agujero" en el punto (3, 6).

6. Funcion Irracional

y = n√R(x)

Una funcion es irracional cuando la variable independiente esta afectada por la operacion de radicacion.

Dominio segun el Indice n

Dada la funcion  y = n√P(x):

• Si n es par: La raiz real existe solo si el radicando es positivo o nulo:
Dom f = { x ∈ Dom P  |  P(x) ≥ 0 }
• Si n es impar: La raiz real existe para cualquier valor real, conservando el signo del radicando:
Dom f = Dom P

y = √x y y = -√x (Ramas de Parabola)

y = 3√x (Definida en R)

7. Ecuaciones Racionales e Irracionales

Ecuaciones Racionales

Una ecuacion racional tiene la forma P(x) / Q(x) = 0. Su resolucion consiste en buscar las raices del numerador P(x) que pertenezcan al dominio de la funcion (es decir, que no anulen a Q(x)).

Ejemplo (PDF): Resolver  (x2 - x - 2) / (x2 + 4x + 3) = 0.
1. Dominio: x2 + 4x + 3 = 0 ⇒ (x + 1)(x + 3) = 0 ⇒ Dom = R ∖ {-1, -3}.
2. Ceros del Numerador: x2 - x - 2 = 0 ⇒ (x - 2)(x + 1) = 0 ⇒ x = 2  ó  x = -1.
3. Filtro de Dominio: El valor x = -1 no pertenece al dominio. La unica solucion es S = {2}.

Ecuaciones Irracionales y Raices Extranas

Para eliminar las raices de una ecuacion irracional, elevamos ambos miembros de la ecuacion a la potencia del indice.

¡IMPORTANTE! Raices Extranas

Al elevar ambos miembros a un exponente par, no siempre se obtiene una ecuacion equivalente. Pueden aparecer soluciones ficticias llamadas raices extranas.
Es obligatorio verificar todas las soluciones en la ecuacion original.

Ejemplo algebraico (PDF): Considere la ecuacion simple:

x + 3 = 5   (Solucion: x = 2)

Si elevamos al cuadrado ambos miembros:

(x + 3)2 = 25  ⇒  x2 + 6x + 9 = 25  ⇒  x2 + 6x - 16 = 0

Resolviendo la cuadratica obtenemos:

x1 = 2   y   x2 = -8

El valor x = -8 es una raiz extrana, pues no verifica la ecuacion original (-8 + 3 ≠ 5).
