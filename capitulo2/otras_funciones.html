2.8 Sistemas de Ecuaciones Lineales

ax + by = c

Un sistema de dos ecuaciones lineales con dos incognitas representa geometricamente dos rectas en el plano cartesiano. Su resolucion consiste en encontrar los puntos en comun.

Definicion Matematica

Un sistema de dos ecuaciones con dos incognitas tiene la forma:

— ax + by = c
— dx + ey = f

Donde a, b, c, d, e, f son coeficientes reales, y x, y son las incognitas.

Clasificacion de Sistemas (Geometria de Rectas)

Compatible Determinado (SCD)

Rectas Oblicuas (se cortan): Tienen una pendiente diferente (m1 ≠ m2).
Existe una solucion unica (el punto de interseccion).

Incompatible (SI)

Rectas Paralelas no Coincidentes: Tienen igual pendiente (m1 = m2) pero distinta ordenada (b1 ≠ b2).
El sistema no tiene solucion.

Compatible Indeterminado (SCI)

Rectas Paralelas Coincidentes: Tienen igual pendiente (m1 = m2) e igual ordenada (b1 = b2). Las dos ecuaciones representan la misma recta. Admite infinitas soluciones (todos los puntos de la recta).

Metodos Analiticos de Resolucion

Existen tres metodos principales para resolver sistemas de ecuaciones de forma analitica. Veremos la explicacion de cada uno resolviendo el sistema:

— x − 2y = 1
— x + y = 4

1. Metodo de Sustitucion

Consiste en despejar una de las incognitas en una ecuacion y sustituir su expresion en la otra ecuacion.

• Despejamos x de la segunda ecuacion:   x = 4 − y
• Sustituimos en la primera ecuacion:   (4 − y) − 2y = 1
• Resolvemos para y:   4 − 3y = 1 ⇒ −3y = −3 ⇒ y = 1
• Reemplazamos y en el despeje de x:   x = 4 − 1 ⇒ x = 3
Solucion: S = {(3, 1)}

2. Metodo de Igualacion

Consiste en despejar la misma incognita en ambas ecuaciones e igualar las expresiones obtenidas.

• Despejamos x en ambas:
   Ecuacion 1: x = 1 + 2y
   Ecuacion 2: x = 4 − y
• Igualamos las expresiones:   1 + 2y = 4 − y
• Agrupamos y resolvemos:   2y + y = 4 − 1 ⇒ 3y = 3 ⇒ y = 1
• Reemplazamos en cualquiera:   x = 4 − 1 ⇒ x = 3
Solucion: S = {(3, 1)}

3. Metodo de Reduccion (Sumas y Restas)

Consiste en sumar o restar las ecuaciones miembro a miembro para eliminar una de las incognitas. Si es necesario, se multiplican previamente por constantes.

• Restamos la primera ecuacion a la segunda para eliminar x:
     (x + y) − (x − 2y) = 4 − 1
     x + y − x + 2y = 3
     3y = 3 ⇒ y = 1
• Reemplazamos en la segunda ecuacion:   x + 1 = 4 ⇒ x = 3
Solucion: S = {(3, 1)}

Resolucion Grafica: Interseccion en el punto (3, 1)

Sistemas con Parametros (Analisis de K)

Frecuentemente nos enfrentamos a sistemas donde algunos coeficientes dependen de un parametro K ∈ R. Analizar el sistema consiste en determinar para que valores de K el sistema es SCD, SCI o SI.

Ejemplo (Ejercicio 60a): Clasificar el sistema segun el valor de K:
— x + y = 2
— Kx + y = 1

Despejamos y en ambas ecuaciones para obtener la ecuacion explicita:
1) y = −x + 2 ⇒ m1 = −1, b1 = 2.
2) y = −Kx + 1 ⇒ m2 = −K, b2 = 1.

Estudio de casos:
• Si K ≠ 1: Las pendientes son distintas (m1 ≠ m2). El sistema es Compatible Determinado (SCD) para todo K ≠ 1.
• Si K = 1: Las pendientes son iguales (m1 = m2 = −1) pero las ordenadas son distintas (b1 = 2 ≠ b2 = 1). El sistema es Incompatible (SI).
• SCI: No existe ningun valor de K que haga b1 = b2, por lo que este sistema nunca podra ser SCI.

Problemas de Aplicacion Analitica

Problema A: Ferrocarriles de Carga (PDF)

El ferrocarril F cobra un fijo de $30 y $8 por tm/km. El ferrocarril F' cobra un fijo de $24 y $8.5 por tm/km. Determinar cual es la zona mas economica para cada uno.

Modelado algebraico:
CF(x) = 8x + 30
CF'(x) = 8.5x + 24

Igualando tarifas:
8x + 30 = 8.5x + 24 ⇒ 6 = 0.5x ⇒ x = 12 km

Conclusion:
• A los 12 km ambos cobran lo mismo ($126).
• Si la distancia es menor a 12 km, es mas economico F'.
• Si la distancia es mayor a 12 km, es mas economico F.

Problema B: Dias de Trabajo de Jose (PDF)

Jose recibe $45 por cada dia trabajado y se le descuentan $27 por cada dia de falta. Luego de 64 dias, recibio $0. ¿Cuantos dias falto?

Sistema de ecuaciones:
x: dias trabajados,   y: dias faltados.
1)   x + y = 64
2)   45x - 27y = 0

Resolucion por sustitucion:
De (2): 45x = 27y ⇒ x = (27/45)y = (3/5)y
En (1): (3/5)y + y = 64 ⇒ (8/5)y = 64 ⇒ y = 40

Conclusion: Jose falto 40 dias (y trabajo 24 dias).

2.9 Funciones Lineales por Tramos

f(x) = tramos

Llamamos funcion definida por tramos (o partes) a aquella que requiere mas de una formula de asignacion segun el intervalo del dominio en el que se encuentre el valor de entrada x.

Caso de estudio analizado

Dada la funcion definida por partes:

f(x) = -x + 2    si   -3 ≤ x < 2
f(x) = -x        si   x ≥ 2

Evaluaciones de ejemplo:
• f(-1): Como -1 ∈ [-3; 2), usamos la regla superior:  f(-1) = -(-1) + 2 = 3.
• f(2): Como 2 ≥ 2, usamos la regla inferior:  f(2) = -(2) = -2.
• f(-4): Como -4 < -3, no hay ninguna regla definida para este valor. Por lo tanto, f(-4) no existe (-4 ∉ Dom f).

Dominio: Dom f = [-3; ∞)
Imagen: Analizando el rango de cada tramo:
— Para [-3; 2): -x + 2 varia en (0; 5].
— Para [2; ∞): -x varia en (-∞; -2].
Union de las imagenes: Im f = (-∞; -2] ∪ (0; 5].

Grafica de la funcion definida por tramos

Los circulos llenos denotan inclusion del extremo en el intervalo, mientras que los circulos vacios representan exclusion.

Funcion Valor Absoluto

y = |x|

La funcion valor absoluto se define por tramos basandose en la definicion algebraica de modulo:

Definicion Matematica

f(x) = |x| =   x     si   x ≥ 0
                -x    si   x < 0

Dominio: R  |  Imagen: [0; ∞)

Modulo Trasladado: f(x) = |x - 3|

Aplicando la definicion de modulo a la expresion interior, obtenemos la siguiente representacion por partes:

f(x) = x - 3     si   x - 3 ≥ 0 ⇒ x ≥ 3
f(x) = -x + 3    si   x - 3 < 0 ⇒ x < 3

Geometricamente, esto traslada la grafica "V" clasica del valor absoluto 3 unidades hacia la derecha, teniendo su vertice en el punto (3, 0).
