Definicion de Funcion

f: A→B

En la practica, el valor de una variable suele depender del de otra (ej. la distancia recorrida depende del tiempo). Representamos esta relacion a traves de una funcion.

Definicion Matematica

Una funcion f: A → B es una correspondencia que a cada elemento x que pertenece al conjunto A le asigna un unico elemento y que pertenece al conjunto B.

Se lee: "f de A en B".
El conjunto A se denomina dominio.
El conjunto B se denomina codominio.

Variable Independiente (x)

x ∈ A

Es la variable a la que se le asignan valores libremente del dominio.

Variable Dependiente (y)

y = f(x) ∈ B

Representa el valor particular que corresponde al valor elegido de x.

Dominio de una funcion

Dom f

Definicion de Dominio

El Dominio (denotado Dom f) es el conjunto de todos los valores admisibles de la variable independiente x para los cuales la formula de la funcion tiene sentido matematico en R.

Las dos restricciones fundamentales del Dominio

Denominador no nulo

f(x) = 1 / (x - 2)

La division por cero no existe. Por tanto, el denominador debe ser distinto de cero:

x - 2 ≠ 0 ⇒ x ≠ 2

Dom f = R - {2}   o   (-∞; 2) ∪ (2; ∞)

Raiz de indice par

f(x) = √(x - 2)

No existen raices pares de numeros negativos en el campo real. El radicando debe ser mayor o igual a cero:

x - 2 ≥ 0 ⇒ x ≥ 2

Dom f = [2; ∞)   o   { x ∈ R / x ≥ 2 }

Imagen de una funcion

Im f

Definicion de Imagen

La Imagen (denotado Im f) es el conjunto de todos los valores resultantes de la variable dependiente y. Es un subconjunto del codominio B que contiene los valores que la funcion realmente toma.

Im f = { y ∈ B / ∃ x ∈ A ∧ f(x) = y }

Donde ∃ significa "existe" y ∧ significa "y".

Diagrama de conjuntos completo: Relacion de conjuntos

La Imagen (Im f) contiene solo los elementos de B que son alcanzados por flechas desde A.

Ejemplos Practicos de Evaluacion

Ejemplo A: Evaluar f(x) = 1 / (x - 2)

Para evaluar la funcion en x = 3, reemplazamos el valor en la formula:

f(3) = 1 / (3 - 2) = 1 / 1 = 1

El resultado es 1. Por lo tanto, el punto (3, 1) pertenece a la grafica de la funcion.

Ejemplo B: Simbologia variable

El nombre de la variable independiente no cambia el significado de la correspondencia. Las siguientes tres expresiones representan la misma relacion de dependencia:

f(x) = 1 / (x - 2)
g(t) = 1 / (t - 2)
h(s) = 1 / (s - 2)
