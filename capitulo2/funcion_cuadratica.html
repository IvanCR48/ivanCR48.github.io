Definicion de la Parabola

y = ax2 + bx + c

La funcion real de segundo grado se conoce como funcion cuadratica y su representacion grafica es una curva simetrica llamada parabola.

Definicion Matematica

Una funcion f: R → R es cuadratica si es de la forma:

f(x) = ax2 + bx + c

Donde los coeficientes a, b, c son numeros reales constantes, con a ≠ 0.

El dominio de esta funcion es siempre el conjunto de los numeros reales: Dom f = R.
La grafica es una parabola de eje de simetria vertical.

3.1 Caracterizacion de los Parametros

El coeficiente cuadratico (a): Concavidad y Abertura

El coeficiente a determina la direccion de apertura y la "abertura" de las ramas de la parabola:
• Si a > 0, la parabola abre hacia arriba (ramas hacia arriba, vertice es un punto minimo).
• Si a < 0, la parabola abre hacia abajo (ramas hacia abajo, vertice es un punto maximo).
• El valor absoluto |a| controla la velocidad de crecimiento: cuanto mayor sea |a|, mas cerrada o angosta sera la parabola; cuanto menor sea |a|, mas abierta o ancha sera.

Efecto del coeficiente 'a' en y = ax2

Traslacion de la Parabola Madre (y = x2)

Podemos dibujar cualquier parabola desplazando la grafica de la parabola base y = x2:

Traslacion Vertical

y = x2 + k

Desplaza la grafica verticalmente |k| unidades. Hacia arriba si k > 0, y hacia abajo si k < 0. El vertice queda en (0, k).

Traslacion Horizontal

y = (x - h)2

Desplaza la grafica horizontalmente |h| unidades. Hacia la derecha si h > 0, y hacia la izquierda si h < 0. El vertice queda en (h, 0).

3.2 Forma Canonica

f(x) = a(x-h)2 + k

La forma canonica de la funcion cuadratica expresa la ecuacion en funcion directa de las coordenadas de su vertice.

Ecuacion Canonica

f(x) = a(x - h)2 + k

Donde el punto V(h, k) (tambien denotado como V(xv, yv)) es el vertice de la parabola.
La recta vertical x = h es el eje de simetria.

Obtencion del Vertice desde la Forma General (ax2 + bx + c)

Para calcular las coordenadas del vertice sin desarrollar el binomio, usamos las formulas:

h = -b / (2a)
k = f(h) = a(h)2 + b(h) + c

Ejemplo Paso a Paso (PDF)

Expresar en forma canonica la funcion cuadratica:  f(x) = x2 + 6x + 5.

1 Identificar coeficientes:
a = 1,   b = 6,   c = 5.
2 Calcular la abscisa del vertice (h):
h = -b / (2a) = -6 / (2 · 1) = -3.
3 Calcular la ordenada del vertice (k):
k = f(-3) = (-3)2 + 6(-3) + 5 = 9 - 18 + 5 = -4.
Por lo tanto, el vertice es V(-3, -4) y el eje de simetria es x = -3.
4 Escribir la forma canonica:
f(x) = a(x - h)2 + k ⇒ f(x) = (x + 3)2 - 4.

3.3 Ceros de la Funcion Cuadratica (Raices)

f(r) = 0

Llamamos raices o ceros de la funcion cuadratica a aquellos valores reales r para los cuales la funcion vale cero. Geometricamente, representan las abscisas de los puntos de interseccion de la parabola con el eje x.

Resolucion de Ecuaciones Incompletas

Sin termino lineal (b = 0)

ax2 + c = 0

Despejamos directamente la variable cuadratica:

x2 = -c / a  ⇒  x = ±√(-c / a)

Ejemplo: 4x2 - 1/4 = 0
x2 = 1/16 ⇒ x = ±1/4

Sin termino independiente (c = 0)

ax2 + bx = 0

Extraemos factor comun x de la expresion:

x(ax + b) = 0  ⇒  x1 = 0,   x2 = -b / a

Ejemplo: 25x2 + 5x = 0
5x(5x + 1) = 0 ⇒ x1 = 0,   x2 = -1/5

Ecuacion Completa y Formula Resolvente

Para una ecuacion completa de la forma ax2 + bx + c = 0 (con a, b, c ≠ 0), utilizamos la formula resolvente (Bhaskara):

x1,2 = [-b ± √(b2 - 4ac)] / (2a)

El radicando Δ = b2 - 4ac se denomina discriminante y determina el numero de soluciones reales:

Discriminante Δ > 0

Dos raices reales distintas

La parabola cruza el eje x en dos puntos distintos.

Discriminante Δ = 0

Una raiz real doble

El vertice de la parabola esta apoyado sobre el eje x.

Discriminante Δ < 0

Sin raices reales

La parabola flota sobre el eje x o esta enteramente por debajo (ej. x2 + 1 = 0).

Ejemplo completo (PDF): Ceros de  f(x) = x2 + 2x - 3.
Coeficientes: a = 1,   b = 2,   c = -3.
Sustitucion:  x = [-2 ± √(22 - 4 · 1 · (-3))] / (2 · 1) = [-2 ± √(4 + 12)] / 2 = [-2 ± 4] / 2.
Las raices resultan: x1 = 1   y   x2 = -3.

Comportamiento de la parabola segun su discriminante (Δ)

3.4 Sistemas Mixtos (Recta y Parabola)

y = mx + b || y = ax^2...

Un sistema mixto esta compuesto por una ecuacion lineal (recta) y una ecuacion cuadratica (parabola). La resolucion analitica se efectua igualando ambas formulas, lo que genera una nueva ecuacion cuadratica.

Clasificacion Geometria (Intersecciones)

Recta Secante

Dos puntos de contacto

La ecuacion igualada tiene discriminante Δ > 0. Hay dos soluciones.

Recta Tangente

Un punto de contacto

La ecuacion igualada tiene discriminante Δ = 0. Toca a la parabola en un solo punto.

Recta Exterior

Sin puntos de contacto

La ecuacion igualada tiene discriminante Δ < 0. La recta no toca a la parabola.

Ejemplo Resuelto (PDF)

Determinar la interseccion entre la parabola  y = x2  y la recta  y = x + 2.

Resolucion analitica (Igualacion):
x2 = x + 2  ⇒  x2 - x - 2 = 0
Factorizando el trinomio:  (x - 2)(x + 1) = 0
Obtenemos dos valores de x:  x1 = 2   y   x2 = -1.

Sustituyendo en la recta para hallar las ordenadas y:
• Para x1 = 2:   y1 = 2 + 2 = 4 ⇒ Punto P1(2, 4).
• Para x2 = -1:   y2 = -1 + 2 = 1 ⇒ Punto P2(-1, 1).

La recta es secante y corta a la parabola en los puntos (2,4) y (-1,1).

3.5 Funciones por Tramos (Cuadraticas)

f(x) = tramos

Las funciones definidas por tramos tambien pueden contener expresiones cuadraticas en sus reglas de correspondencia. Su analisis de continuidad se evalua en las uniones de los intervalos.

Caso de estudio practico (Ejercicio 66b del PDF)

Dada la funcion real definida por partes:

f(x) = 4            si   x < -2
f(x) = x2 - 1     si   -2 ≤ x ≤ 2
f(x) = 3            si   x > 2

Analisis de Dominio e Imagen:
• Dominio: Los tres intervalos abarcan la totalidad de la recta real. Por tanto, Dom f = R.
• Imagen:
— Para x < -2: la funcion es constante e igual a 4.
— Para -2 ≤ x ≤ 2: la parabola x2 - 1 tiene su vertice en (0, -1) y sus extremos en f(±2) = 3. Varia en el intervalo [-1; 3].
— Para x > 2: la funcion es constante e igual a 3.
Union de las imagenes: Im f = [-1; 3] ∪ {4} ⇒ [-1; 4] (excluyendo el tramo intermedio (3; 4)).

Grafica de la funcion cuadratica por tramos

En x = -2 hay un salto discontinuo de 4 a 3. En x = 2, el tramo cuadratico y el constante se acoplan continuamente en y = 3.
