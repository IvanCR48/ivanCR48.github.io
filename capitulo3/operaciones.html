1. Operaciones y Definicion

(f + g)(x)

Del mismo modo que podemos realizar operaciones basicas con numeros reales, es posible combinar dos o mas funciones para obtener nuevas funciones a traves del algebra de funciones.

Definicion Matematica

Sean f y g dos funciones reales. Se definen las funciones resultantes como:

  • • Suma:  (f + g)(x) = f(x) + g(x)
  • • Diferencia:  (f − g)(x) = f(x) − g(x)
  • • Producto:  (f · g)(x) = f(x) · g(x)
  • • Cociente:  (f / g)(x) = f(x) / g(x)  (siempre que g(x) ≠ 0)

Dominio de la Funcion Resultante

Para que la operacion tenga sentido, la variable independiente x debe pertenecer simultaneamente al dominio de f y al dominio de g.

La Regla de la Interseccion

El dominio de la suma, diferencia y producto es la interseccion de los dominios individuales:

Dom(f + g) = Dom(f − g) = Dom(f · g) = Dom f ∩ Dom g

Para el cociente, debemos excluir ademas todos los valores de x que anulan al denominador g(x):

Dom(f / g) = (Dom f ∩ Dom g) ∖ { x ∈ R  |  g(x) = 0 }

Ejemplos de Aplicacion Analitica

Ejemplo 1 (Polinómicas)

Sean  f(x) = 3x2 + 2  y  g(x) = 5x − 4.
Ambas son funciones polinomicas, por lo que:  Dom f = R  y  Dom g = R.

1) Suma:
(f + g)(x) = (3x2 + 2) + (5x − 4) = 3x2 + 5x − 2
Dom(f + g) = R ∩ R = R

2) Diferencia:
(f − g)(x) = (3x2 + 2) − (5x − 4) = 3x2 − 5x + 6
Dom(f − g) = R

3) Producto:
(f · g)(x) = (3x2 + 2)(5x − 4) = 15x3 − 12x2 + 10x − 8
Dom(f · g) = R

4) Cociente:
(f / g)(x) = (3x2 + 2) / (5x − 4)
El denominador se anula si:   5x − 4 = 0 ⇒ x = 4/5.
Dom(f / g) = R ∖ {4/5}

Ejemplo 2 (Radical y Racional)

Sean  f(x) = √x  y  g(x) = 1 / (x2 − 5x + 6).
Analicemos el producto  (f · g)(x)  y su dominio resultante.

1. Dominios individuales:
• f(x) tiene indice par (raiz cuadrada) ⇒ radicando ≥ 0 ⇒ Dom f = [0; ∞).
• g(x) tiene denominador ⇒ x2 − 5x + 6 ≠ 0 ⇒ (x − 2)(x − 3) ≠ 0 ⇒ Dom g = R ∖ {2, 3}.

2. Interseccion:
Dom(f · g) = Dom f ∩ Dom g = [0; ∞) ∩ (R ∖ {2, 3})
Dom(f · g) = [0; 2) ∪ (2; 3) ∪ (3; ∞)

Ejercicios de Autoevaluacion

Dadas las siguientes funciones del modulo:
  • • f(x) = x / (x2 − 2x − 3)  ⇒  Dom f = R ∖ {−1, 3}
  • • g(x) = x / (x − 2)  ⇒  Dom g = R ∖ {2}
  • • h(x) = 5  ⇒  Dom h = R
  • • s(x) = 1 / x  ⇒  Dom s = R ∖ {0}
  • • r(x) = √(x − 3)  ⇒  Dom r = [3; ∞)
  • • t(x) = √(x + 2)  ⇒  Dom t = [−2; ∞)
Ejercicio 154-a: Calcular (r + g)(2)

¿Existe el resultado de sumar las funciones en el valor x = 2?

Ver Resolucion

Para evaluar (r + g)(2), x = 2 debe pertenecer al dominio resultante:
Dom(r + g) = Dom r ∩ Dom g.
• Dom r = [3; ∞) ⇒ 2 ∉ Dom r.
• Dom g = R ∖ {2} ⇒ 2 ∉ Dom g.
Como el valor x = 2 no pertenece a ninguno de los dominios individuales (ni a su interseccion), la operacion no existe para este valor.

Ejercicio 154-b: Calcular (h · s)(−6)

Evaluar el producto de las funciones en el valor x = −6.

Ver Resolucion

Verificamos el dominio: −6 ∈ (Dom h ∩ Dom s) = R ∖ {0}. Es un punto valido.
Calculamos individualmente:
• h(−6) = 5
• s(−6) = 1/(−6) = −1/6
Operamos:
(h · s)(−6) = 5 · (−1/6) = −5/6

Ejercicio 154-c: Calcular (g / f)(4)

Evaluar el cociente de las funciones en el valor x = 4.

Ver Resolucion

Verificamos el dominio: 4 ∈ (Dom g ∩ Dom f) = R ∖ {−1, 2, 3}.
Calculamos individualmente:
• g(4) = 4 / (4 − 2) = 4/2 = 2
• f(4) = 4 / (42 − 2·4 − 3) = 4 / (16 − 8 − 3) = 4/5
Operamos:
(g / f)(4) = g(4) / f(4) = 2 / (4/5) = 10/4 = 5/2 (ó 2.5)

Ejercicio 154-d: Calcular (h / r)(−1)

Evaluar el cociente de las funciones en el valor x = −1.

Ver Resolucion

Verificamos si x = −1 pertenece al Dom(h/r):
Dom(h/r) = (Dom h ∩ Dom r) ∖ {x | r(x) = 0}
• Dom h = R
• Dom r = [3; ∞) ⇒ −1 ∉ Dom r.
Dado que −1 no pertenece al dominio de r, la division no existe para este valor.
