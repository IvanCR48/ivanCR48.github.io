2. Definicion

(f ° g)(x)

Existe otra forma de obtener una nueva funcion a partir de dos dadas: evaluando una funcion en el resultado de la otra. Esto se conoce como composicion de funciones.

Definicion Matematica

Dadas dos funciones f y g, la funcion compuesta, denotada por f ° g (se lee "f compuesta con g"), esta definida por:

(f ° g)(x) = f(g(x))

Donde g(x) es la funcion interna (se aplica primero) y f(y) es la funcion externa (se aplica al resultado de g).

Diagrama de Flujo del Encadenamiento (f ° g)(x)

2.1 Dominio de la Funcion Compuesta

El dominio de la composicion es el conjunto de todas las entradas del dominio de la funcion interna g tales que su resultado pertenezca al dominio de la externa f.

Definicion del Dominio

El dominio de f ° g es el conjunto:

Dom(f ° g) = { x ∈ Dom g  |  g(x) ∈ Dom f }

Esto significa que para poder calcular f(g(x)) se deben cumplir dos filtros secuenciales:
1) x debe poder ingresar a g (es decir, x ∈ Dom g).
2) El valor de salida g(x) debe poder ingresar a f (es decir, g(x) ∈ Dom f).

¡ADVERTENCIA MUY IMPORTANTE!

No es suficiente mirar la formula final simplificada para determinar el dominio de una composicion. Si un valor de x anula o rompe la funcion interna g(x), ese valor nunca podra pertenecer a Dom(f ° g), aun si la formula final simplificada no muestra ningun problema.

Ejemplos Explicados Paso a Paso

Ejemplo 1: Polinómicas

Sean  f(x) = 5x2 − 3x + 2  y  g(x) = 4x + 3.
Ambas tienen dominio real R.

a) Evaluar (f ° g)(−2):
• Primero calculamos la funcion interna:  g(−2) = 4(−2) + 3 = −5.
• Evaluamos f en ese resultado:  f(−5) = 5(−5)2 − 3(−5) + 2 = 125 + 15 + 2 = 142.

b) Evaluar (g ° f)(2):
• Primero la interna:  f(2) = 5(2)2 − 3(2) + 2 = 20 − 6 + 2 = 16.
• Evaluamos g:  g(16) = 4(16) + 3 = 64 + 3 = 67.

c) Formula de (f ° g)(x):
(f ° g)(x) = f(4x+3) = 5(4x+3)2 − 3(4x+3) + 2
             = 5(16x2 + 24x + 9) − 12x − 9 + 2
             = 80x2 + 108x + 38   ⇒  Dom = R.

d) Formula de (g ° f)(x):
(g ° f)(x) = g(5x2−3x+2) = 4(5x2−3x+2) + 3
             = 20x2 − 12x + 11   ⇒  Dom = R.

Conclusion: Dado que 80x2+108x+38 ≠ 20x2-12x+11, se demuestra que la composicion no es conmutativa:  f ° g ≠ g ° f.

Ejemplo 2: Radical y Lineal

Sean  f(x) = √x  (Dom f = [0; ∞))  y  g(x) = 2x − 3  (Dom g = R).

Composicion (f ° g)(x):
(f ° g)(x) = f(g(x)) = √(2x − 3)

Calculo del Dominio:
Buscamos los x ∈ Dom g (reales) tales que g(x) ∈ Dom f:
• 2x − 3 ≥ 0 ⇒ 2x ≥ 3 ⇒ x ≥ 3/2.
Dom(f ° g) = [3/2; ∞).

Ejemplo 3: Racionales (El Peligro de Simplificar)

Sean  f(x) = x / (x + 1)  (Dom f = R ∖ {−1})  y  g(x) = 2 / (x − 1)  (Dom g = R ∖ {1}).

a) Analisis de (f ° g)(x):
(f ° g)(x) = f(2 / (x−1)) = [ 2/(x−1) ] / [ 2/(x−1) + 1 ]
Multiplicando numerador y denominador por (x−1) para simplificar la fraccion compleja:
(f ° g)(x) = 2 / [ 2 + (x − 1) ] = 2 / (x + 1)

¿Cual es el dominio de f ° g?
• Si miramos la formula final 2 / (x + 1), pareceria que solo debemos excluir x = −1.
• Sin embargo, x debe pertenecer al dominio de la interna g, por lo que x = 1 debe ser excluido desde el inicio.
⇒ Dom(f ° g) = R ∖ {−1, 1}.

b) Analisis de (g ° f)(x):
(g ° f)(x) = g(x/(x+1)) = 2 / [ x/(x+1) − 1 ]
Multiplicando por (x+1) para simplificar:
(g ° f)(x) = 2(x+1) / [ x − (x + 1) ] = 2(x+1) / (−1) = −2x − 2

¿Cual es el dominio de g ° f?
• La formula final es lineal (−2x − 2), cuyo dominio natural son todos los reales.
• No obstante, la entrada x debe ingresar primero a la interna f, por lo que x = −1 debe excluirse de forma obligatoria.
⇒ Dom(g ° f) = R ∖ {−1}.

Ejemplo 4: Aplicacion Practica (Mundial 2026 - Globo de Fuego)

Supongamos que el radio r (en centimetros) de un globo circular de helio aumenta con el tiempo t (en segundos) segun la funcion:
 r(t) = 5t + 3   (para 0 ≤ t ≤ 8).

(a) Area del circulo como funcion de t:
El area de un circulo es  A(r) = πr2.
Componiendo obtenemos la funcion de area respecto al tiempo t:
A(t) = (A ° r)(t) = π(5t + 3)2.

(b) Determinar el area a los 6 segundos (t = 6):
A(6) = π(5(6) + 3)2 = π(33)2 = 1089π cm2  (≈ 3421.19 cm2).

(c) Tiempo requerido para que el area sea de 324π cm2:
Igualamos la funcion de area:  π(5t + 3)2 = 324π
⇒ (5t + 3)2 = 324
Aplicamos raiz cuadrada en ambos miembros:   5t + 3 = 18   (descartamos el resultado negativo ya que t ≥ 0)
⇒ 5t = 15 ⇒ t = 3 segundos.

Ejercicio 154-e: Calcular (f ° g)(4)

Evaluar la composicion en el valor x = 4.

Ver Resolucion

Verificamos dominios:
• x = 4 pertenece a Dom g (R ∖ {2}).
• Calculamos la interna:  g(4) = 4 / (4 − 2) = 4/2 = 2.
• Verificamos si g(4)=2 pertenece a Dom f (R ∖ {−1, 3}). Es correcto.
• Evaluamos externa:  f(2) = 2 / (22 − 2·2 − 3) = 2 / (4 − 4 − 3) = −2/3.

Ejercicio 154-f: Calcular (h ° s)(−6)

Evaluar la composicion en el valor x = −6.

Ver Resolucion

Verificamos dominios:
• −6 pertenece a Dom s (R ∖ {0}).
• Interna:  s(−6) = −1/6.
• Evaluamos externa h (funcion constante):  h(−1/6) = 5.

Ejercicio 155-h: Determinar formula y dominio de (r ° g)(x)

Componer las funciones radical y racional analiticamente.

Ver Resolucion

1. Formula:
(r ° g)(x) = r(g(x)) = √[ x/(x−2) − 3 ]
Efectuando la resta dentro de la raiz:
(r ° g)(x) = √[ (x − 3(x − 2)) / (x − 2) ] = √[ (6 − 2x) / (x − 2) ].

2. Dominio:
Debemos asegurar que:
a) x ∈ Dom g ⇒ x ≠ 2.
b) El radicando sea no negativo:   (6 − 2x) / (x − 2) ≥ 0.
Hacemos tabla de signos para la inecuacion racional con los puntos criticos x=2 y x=3:
• Si x < 2: (6-2x) es positivo (+) y (x-2) es negativo (-) ⇒ Cociente negativo (-).
• Si 2 < x ≤ 3: (6-2x) es positivo (+) y (x-2) es positivo (+) ⇒ Cociente positivo (+).
• Si x > 3: (6-2x) es negativo (-) y (x-2) es positivo (+) ⇒ Cociente negativo (-).

Por lo tanto, la inecuacion se satisface en el intervalo: (2; 3] (abierto en 2 por el denominador, cerrado en 3 por la raiz).
Dom(r ° g) = (2; 3].
