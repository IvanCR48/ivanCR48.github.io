3. Funcion Par e Impar

La simetria es una propiedad geometrica intrinseca de ciertas funciones que simplifica su estudio y representacion grafica.

3.1.1 Función Par

Una función f es par si para cada x en su dominio, −x también pertenece al dominio y se cumple:

f(−x) = f(x)

Simetría geométrica: Su gráfica es simétrica respecto al eje y (actúa como un espejo).
Ejemplo: f(x) = x2 es par porque:
f(−x) = (−x)2 = x2 = f(x).

3.1.2 Función Impar

Una función f es impar si para cada x en su dominio, −x también pertenece al dominio y se cumple:

f(−x) = −f(x)

Simetría geométrica: Su gráfica es simétrica respecto al origen (0,0) (rotación de 180°).
Ejemplo: f(x) = x3 es impar porque:
f(−x) = (−x)3 = −x3 = −f(x).

4. Función Inyectiva, Sobreyectiva y Biyectiva

Dada una función f: A → B, se clasifica de acuerdo a cómo se relacionan los elementos del dominio A con el codominio B.

Definiciones Formales

  • 1. Función Inyectiva (Uno a Uno):
    Elementos distintos del dominio tienen imágenes distintas.
    f es inyectiva ⇒ (x ≠ y ⇒ f(x) ≠ f(y)), o equivalente: f(x) = f(y) ⇒ x = y.
  • 2. Función Sobreyectiva o Suryectiva (Sobre):
    Todos los elementos del codominio B son imagen de al menos un elemento del dominio.
    f es sobreyectiva ⇒ Im f = B.
  • 3. Función Biyectiva:
    Es inyectiva y sobreyectiva simultáneamente. Esto asegura un emparejamiento perfecto uno a uno.
Test de la Recta Horizontal (Inyectividad):

Si cualquier recta horizontal interseca a la gráfica de una función en a lo sumo un punto, entonces la función es inyectiva. Si la interseca en dos o más puntos, la función no es inyectiva.

5. Función Inversa

La funcion inversa nos permite desandar el camino de la funcion original y recuperar los valores de entrada a partir de las salidas.

5.1 Definicion de Funcion Inversa

Sea f: A → B una función biyectiva. Se llama función inversa de f, denotada por f−1, a la función f−1: B → A tal que:

f−1(y) = x   ⇒   f(x) = y

5.2 Propiedades Clave:
1) Identidad:  (f ° f−1)(x) = x   y   (f−1 ° f)(x) = x.
2) Simetría Gráfica: Las gráficas de f y f−1 son **simétricas con respecto a la recta y = x**. Cada punto (x, y) de la gráfica de f corresponde a un punto (y, x) de la gráfica de f−1.

Simetría de la Función Inversa respecto a y = x

Desarrollos Analíticos del Módulo

Ejemplo 1 (Lineal):
Dada la función biyectiva:  f: R → R  /  f(x) = 2x − 4.
1. Escribimos:  y = 2x − 4.
2. Intercambiamos variables:  x = 2y − 4.
3. Despejamos y:  x + 4 = 2y ⇒ y = x/2 + 2.
Por lo tanto:  f−1(x) = x/2 + 2  con dominio e imagen en R.
Ejemplo 2 (Cuadrática con Restricción):
Dada  f(x) = x2 − 3. No es biyectiva en R (falla el test horizontal).
• Paso 1 (Restringir): Consideramos el dominio a valores no negativos: f: [0; ∞) → [−3; ∞). Ahora es biyectiva.
• Paso 2 (Invertir): Escribimos  y = x2 − 3  ⇒  intercambiamos:  x = y2 − 3.
• Paso 3 (Despejar): y2 = x + 3 ⇒ y = √(x + 3)  (seleccionamos la raíz positiva ya que la imagen de la inversa debe ser el dominio original [0; ∞)).
Por lo tanto:  f−1(x) = √(x + 3)  con dominio [−3; ∞) e imagen [0; ∞).
Ejemplo 3 (Completación de Cuadrados):
Dada la función cuadrática completa:  f(x) = x2 + 4x + 1.
Restringimos para hacerla biyectiva:  f: [−2; ∞) → [−3; ∞).
• Paso 1 (Completar cuadrados):
y = x2 + 4x + 1 = (x2 + 4x + 4) − 4 + 1 = (x + 2)2 − 3.
• Paso 2 (Intercambiar y Despejar):
x = (y + 2)2 − 3 ⇒ x + 3 = (y + 2)2
⇒ y + 2 = √(x + 3) ⇒ y = √(x + 3) − 2.
Por lo tanto:  f−1(x) = √(x + 3) − 2  con Dom = [−3; ∞) e Im = [−2; ∞).

3.2 Ejercicios Prácticos del Módulo (Ejercicio 163)

Clasifica cada una de las siguientes funciones en Inyectiva (I), Suryectiva (S) o Biyectiva (B).

Ejercicio 163-a

f: R → R  /  f(x) = −3x + 1

Ver Clasificación

Clasificación: Biyectiva (B)
• Es una función lineal con pendiente no nula (−3).
• Inyectividad: −3x1 + 1 = −3x2 + 1 ⇒ x1 = x2 (Cumple).
• Suryectividad: Su imagen son todos los reales, que coincide con el codominio R (Cumple).

Ejercicio 163-b

f: R → R  /  f(x) = x2

Ver Clasificación

Clasificación: Ninguna
• Inyectividad: No lo es, pues f(−2) = f(2) = 4 (dos valores de entrada distintos dan el mismo resultado).
• Suryectividad: No lo es, ya que su imagen es [0; ∞), el cual no es igual al codominio R (los negativos no tienen preimagen).

Ejercicio 163-c

f: R → R  /  f(x) = x3

Ver Clasificación

Clasificación: Biyectiva (B)
• Inyectividad: Las raíces cúbicas de números reales son únicas, x13 = x23 ⇒ x1 = x2 (Cumple).
• Suryectividad: Cualquier número real tiene una raíz cúbica real, por lo que la imagen es R (Cumple).

Ejercicio 163-d

f: [0; ∞) → R  /  f(x) = x2

Ver Clasificación

Clasificación: Inyectiva (I)
• Inyectividad: Al estar restringido a valores positivos y cero, no existen dos valores distintos con la misma imagen.
• Suryectividad: No lo es, ya que el codominio es R y la imagen es [0; ∞).
